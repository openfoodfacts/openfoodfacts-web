Comment fonctionne le jeu

Le jeu affiche un produit aléatoire dont certaines données sont manquantes. Répondez "Oui / Non / Passer" par rapports aux caractéristiques pour compléter les données !

L'interface se scinde en trois parties :

1. La question

Répondez en cliquant sur "Oui", "Non", ou "Passer". Le jeu passe automatiquement au produit suivant.

Utilisez les filtres en haut de l'écran pour vous concentrer sur des questions spécifiques :

  • Label : Est-ce que le produit a un label spécifique (ex : "Bio", "Fait Trade") ?
  • Catégorie : Est-ce qu'il appartient à une catégorie (ex : "Yaourt", "Cookies au chocolat") ?
  • Marque : Questions à propos de la marque.
  • Poids du produit : Questions à propos du poids et du volume du produit (ex : 100g, 1L).

Ce qui va s'afficher :

  1. Une boîte de recherche (optionnelle) pour filtrer par marque, label, etc.
  2. La question (ex : "Est-ce que le produit correspond à la description ?").
  3. Une photo du produit.
  4. Les trois buttons (Oui / Non / Passer).

Important: Si vous n'êtes pas sûr(e) à 100%, choisissez "Passer" ! La précision est importante. Quelqu'un d'autre pourra vérifier les informations plus tard. La collaboration rend les données meilleures !

2. Accès à l'information

Si la réponse n'est pas claire sur l'image, utilisez ces outils :

  • Cliquez sur le produit pour zoomer.
  • Utilisez le bouton "Voir" pour voir la page entière du produit sur Open Food Facts.
  • Utilisez le bouton "Modifier" pour corriger la page du produit (utile pour les modifications complexes).

3. Annotations et progression

A droite (ou en bas sur téléphone), vous trouverez des informations utiles :

  • Nombre de produits restant (selon les filtres utilisés).
  • Le nombre de contributions pour la session en cours.
  • Un historique de vos contributions récentes.
