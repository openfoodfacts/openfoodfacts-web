Fakta Pangan Terbuka ing riset: publikasi, kutipan, lan sumber daya kanggo peneliti

Wiwit taun 2012, Open Food Facts wis dadi sumber daya sing penting kanggo para peneliti ing saindenging jagad sing nggunakake data pangan kanggo panliten babagan nutrisi, lingkungan, kesehatan masyarakat, lan akeh bidang liyane. Basis data terbuka kita wis dikutip ing atusan makalah ilmiah.

Wilayah Riset Utama

Data kita ndhukung maneka warna domain riset

Nutrisi & Kesehatan Masyarakat

Para panaliti nggunakake basis data kita kanggo nyinaoni kualitas nutrisi, komposisi panganan, lan intervensi kesehatan masyarakat. Data Open Food Facts wis dadi instrumen penting ing riset lan evaluasi Nutri-Score.

Dampak Lingkungan

Panliten babagan jejak karbon, Eco-Score, lan kelestarian lingkungan produk panganan entuk manfaat saka database informasi produk sing lengkap.

Pangolahan Pangan

Riset babagan panganan ultra-olahan nggunakake klasifikasi NOVA gumantung marang dhaptar bahan lan informasi pangolahan sing rinci.

Ilmu Komputer & AI

Pembelajaran mesin, pangenalan gambar, pangolahan basa alami, lan proyèk ilmu data migunakaké set data lan gambar sing mbukak.

Prilaku Konsumen

Panliten sing nliti pilihan panganan, efek label, lan transparansi ing industri panganan nggunakake data Open Food Facts.

Kebijakan Pangan

Para pembuat kebijakan lan peneliti nganalisa peraturan pangan, syarat pelabelan, lan praktik industri nggunakake basis data transparan kita.

Sumber Daya kanggo Para Peneliti

Cara nggunakake lan nyutat Fakta Panganan Terbuka

Ngakses Data

Open Food Facts nyedhiyakake pirang-pirang cara kanggo ngakses data kita:

  • Ekspor basis data: Unduh dump basis data lengkap ing macem-macem format. Deloken kaca Data kanggo katrangan luwih lanjut.
  • API: Nliti database kanthi program liwat API lengkap. Dokumentasi API
  • Antarmuka telusuran: Jelajahi produk langsung ing situs web kita kanthi kemampuan telusuran lan penyaringan sing luwih canggih.

Ngutip Fakta Panganan Terbuka

Menawi panjenengan ngginakaken Open Food Facts ing riset panjenengan, sumangga dipunsebataken kanthi trep. Iki format kutipan sing disaranake:

Kutipan situs web:

Fakta Panganan Terbuka. (Taun). Fakta Panganan Terbuka - Donya. Dijupuk saka https://world.openfoodfacts.org


Kutipan set data:

Fakta Panganan Terbuka. (Taun). Basis Data Fakta Pangan sing Terbuka [Set data]. Dijupuk saka https://world.openfoodfacts.org/data

Lisensi

Data kita mbukak lan bisa digunakake maneh kanthi bebas:

  • Basis data iki kasedhiya ing sangisore Lisensi Basis Data Terbuka (ODbL)
  • Isi basis data ana ing sangisore Lisensi Isi Basis Data (DbCL)
  • Gambar produk kasedhiya ing Creative Commons Attribution ShareAlike (CC BY-SA 3.0)

See our Terms and conditions of use and re-use for complete details.

Data Quality

Minangka basis data crowdsourced, kualitas data bisa beda-beda. Nalika nggunakake Open Food Facts kanggo riset:

  • Coba saring miturut skor kelengkapan utawa indikator kualitas data
  • Validasi titik data penting yen presisi iku penting
  • Waspada marang bias geografis lan temporal ing jangkoan data kita
  • Hubungi kita yen sampeyan butuh bantuan babagan penilaian kualitas data kanggo riset sampeyan

Kolaborasi karo Kita

Kita nampani kerjasama karo tim riset:

  • Kebutuhan data: Kabari kita yen sampeyan butuh data tartamtu utawa perbaikan struktur basis data kita
  • Kelompok pemindai: Atur kampanye pangumpulan data sing ditargetake kanggo wilayah riset utawa kategori produk sampeyan
  • Nuduhake asil: Nuduhake panemuanmu karo komunitas kita lan tulungi ningkatake kawruh
  • Pangembangan piranti: Kolaborasi ing piranti sing migunani kanggo peneliti lan konsumen

Hubungi kita ing contact@openfoodfacts.org kanggo ngrembug babagan kesempatan kolaborasi.

Dampak Riset

Open Food Facts wis menehi kontribusi sing signifikan kanggo riset ilmiah lan kesehatan masyarakat:

  • Luwih saka 600 publikasi ilmiah wis ngutip basis data kita
  • Tim riset utama kalebu EREN (Prancis), USP (Brasil), lan Universitas Liège (Belgia) wis nggunakake data kita
  • Basis data kita wis ndhukung riset sing ndadékaké owah-owahan kabijakan lan rekomendasi kesehatan masyarakat
  • Kita wis ngaktifake perbandingan lintas negara lan studi global babagan kualitas lan nutrisi panganan.
