A base de dados de produtos cosméticos

O Open Beauty Facts é uma base de dados de produtos cosméticos com ingredientes, alergénios, indícios e todas as e todas as informações que podemos encontrar nos rótulos dos produtos.

Feito por todos

O Open Beauty Facts é uma associação sem fins lucrativos de voluntários.

Mais de 1800 colaboradores adicionaram mais de 13.000 produtos de 134 países usando a nossa aplicação , iPhone ou Windows Phone ou as suas câmaras para ler códigos de barras e enviar fotografias de produtos e os seus respetivos rótulos.

Para todos

Os dados sobre cosméticos são de interesse público e devem ser abertos. A base de dados completa é publicada como dados abertos e pode ser reutilizada por qualquer pessoa e para qualquer utilização. Veja as reutilizações interessantes ou faça suas próprias!

Cosméticos, para inovação e ciência

Os nossos dados podem ser reutilizados para educação, novos produtos e serviços, pesquisas e muito mais!

A base de dados do Open Beauty Facts são dados abertos publicados sob a Open Database License. Qualquer pessoa pode usá-lo para qualquer finalidade - comercial ou não comercial - desde que a fonte dos dados seja atribuída e as melhorias na base de dados sejam partilhadas de maneira semelhante.

Educação

Quantos anúncios de cosméticos viu na sua vida? Os dados abertos podem ajudar a ver os factos sob o marketing.

Produtos e serviços

Os dados do Open Beauty Facts podem ser reutilizados em projetos não comerciais e comerciais, desde que as melhorias na base de dados também sejam partilhadas.

Ciência

Os dados do Open Beauty Facts são utilizados por cientistas em diferentes áreas (Química, Biologia, Dados Abertos Vinculados..). E não precisa de um doutoramento para participar da ciência cidadã!
