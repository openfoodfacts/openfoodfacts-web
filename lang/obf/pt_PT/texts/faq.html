Como posso entrar em contacto com a equipa do Open Beauty Facts, fazer perguntas ou fazer sugestões?

Pode fazer perguntas ou fazer sugestões no fórum de ideias ou entre em contacto connosco pelo e-mail: contact@openbeautyfacts.org

As informações e dados sobre produtos são verificados?

As informações e os dados são enviados pelos colaboradores do Open Beauty Facts. Os colaboradores também enviam imagens do produto, rótulos, listas de ingredientes e alegações de saúde e beleza. Em caso de dúvida, os visitantes podem verificar a precisão por si mesmos e, se houver um erro, eles poderão corrigi-lo.

Para detetar erros potenciais com mais facilidade, vamos adicionar progressivamente verificações automatizadas. por exemplo, se os ingredientes de um produto são muito diferentes dos produtos da mesma categoria, isso pode ser um erro.

Posso adicionar imagens de produtos ou dados do site do fabricante, sites de compras ou outros sites?

Quase todos os outros sites proíbem a reprodução e a reutilização dos seus dados e imagens e essa é a razão pela qual estamos a criar o Open Beauty Facts: para disponibilizar todos esses dados para todos e para todas as utilizações.

Para evitar qualquer problema legal, pedimos aos colaboradores que adicionem apenas fotografias que eles mesmos tiraram e apenas os dados provenientes da embalagem e do rótulo do produto.

Sou um fabricante de produtos cosméticos, posso adicionar os meus próprios produtos?

Sim! A única condição é aceitar que os dados e as imagens sejam disponibilizados sob uma licença aberta. (veja os Termos de contribuição)

Se tiver um grande número de produtos para adicionar (mais de 50), podemos tentar encontrar uma maneira de adicionar automaticamente os produtos e os seus dados. Contacte-nos para discutir como podemos prosseguir: contact@openbeautyfacts.org

Qual é a diferença com outros sites, serviços e aplicações móveis que já permitem visualizar informações sobre produtos cosméticos?

A diferença principal para nós é uma crítica: os nossos dados estão disponíveis gratuitamente para todos e para todas as utilizações. É o que chamamos dados abertos.

Quase todos os outros sites, serviços e aplicações proíbem que outros reproduzam e reutilizem os seus dados. Muito pelo contrário, guardam-nos ciosamente para eles mesmos. Em quase todos os casos, os termos de serviço proíbem explicitamente qualquer uso não pessoal e qualquer extração de todos os dados ou partes dos dados.

Consideramos que a informação sobre produtos cosméticos é importante e útil demais para mantê-la trancada num cofre. Por isso, decidimos fazer exatamente o contrário: não apenas permitimos o uso e a reutilização da nossa base de dados, gratuitamente e sem custos, para todos e para todas as utilizações (inclusive comerciais), mas também o encorajamos!

Tornando os dados disponíveis publicamente (o que é conhecido como dados abertos) permite indivíduos, associações, empresas, pesquisadores, etc. de todo o mundo para pensar e desenvolver aplicações para os dados que certamente nunca teríamos pensado.

What is Open Beauty facts’ economical model

Os colaboradores são voluntários. As suas contribuições são reunidas numa base de dados aberta que pode ser utilizada por todos e para todas as utilizações. (veja os Termos de reutilização)

Todos (incluindo, entre outros, os colaboradores e criadores do Open Beauty Facts) podem, assim redistribuir e/ou reutilizar os dados para criar sites, serviços, programas informáticos, aplicações móveis ou para escrever artigos e estudos. Eles podem disponibilizar gratuitamente o trabalho resultante ou vender ou gerar receitas (por exemplo, com anúncios), desde que respeitem os termos de reutilização.
