Open Beauty Facts é uma base de dados de produtos cosméticos criada por todos, para todos. Pode utilizá-lo para fazer melhores escolhas em termos de cuidados pessoais e, como se trata de dados abertos, qualquer pessoa pode reutilizá-los para qualquer finalidade.

Contribuir

O Open Beauty Facts é um projeto sem fins lucrativos desenvolvido por milhares de voluntários de todo o mundo. Pode começar a contribuir adicionando um produto da sua casa de banho com a nossa aplicação para iPhone ou Android, e temos muitos projetos interessantes para os quais pode contribuir de várias maneiras.
