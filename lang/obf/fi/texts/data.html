Tiedot

Open Beauty Facts -tietokanta on saatavilla Open Database -lisenssillä.
Tietokannan yksittäinen sisältö on saatavilla Tietokannan sisältölisenssillä.
Tuotteen kuvat ovat saatavilla Creative Commons Attribution ShareAlike -lisenssillä. Ne saattavat sisältää tekijänoikeuden tai muun oikeuden alaisia graafisia elementtejä, jotka joissain tapauksissa saa kopioida (lainausoikeudet tai kohtuullinen käyttö).

Lue Käyttöehdot ja uudelleenkäyttö ennen tietojen uudelleenkäyttöä.

Olemme erittäin kiinnostuneita oppimaan, mihin Open Beauty Facts -tietoja käytetään. Se ei ole pakollista, mutta olisimme erittäin kiitollisia, jos kerrot meille uudelleenkäytöstäsi jotta voimme jakaa ne Open Beauty Facts -yhteisön kanssa.

Muutosten vienti

Päivittäiset muutosten viennit ovat saatavilla viimeiseltä 14:ltä päivältä. Luettelo tällä hetkellä saatavilla olevista delta-tiedostoista löytyy osoitteesta https://static.openbeautyfacts.org/data/delta/index.txt. Jokainen rivi edustaa tiedostoa, joka on saatavilla osoitteessa https://static.openbeautyfacts.org/data/delta/{filename}. Tiedostonnimi sisältää UNIX-aikaleiman ensimmäisestä ja viimeisestä muutoksesta, jotka sisältyvät JSON-tiedostoon, jotta muutostiedostot voidaan tuoda (purkamisen jälkeen) aakkosjärjestyksessä komennolla mongoimport.

Live JSON API

API käyttäytyy 99-prosenttisesti kuten Open Food Facts, joten monien Open Food Facts -sovellusliittymän kanssa vuorovaikutuksessa olevien SDK-koodien pitäisi toimia vaihtamalla päätepiste openfoodfacts.orgista openbeautyfacts.org:iin. Koodia ei ole päivitetty yli vuoteen, joten monet uudemmat sovellusliittymät, kuten Knowledge Panels, eivät ole vielä saatavilla.

Myös JSON API on saatavilla tuotteen tietojen lukemiseen. Tätä API:ta käytetään erityisesti Open Beauty Facts -mobiilisovelluksessa iPhonelle ja Androidille.

Muutamme API:a ajan myötä, mutta huolehdimme yhteensopivuudesta vanhojen versioiden kanssa. Kerro meille, jos käytät sitä, jotta voimme kertoa sinulle mahdollisista muutoksista.

URL-osoite tuotteen tietojen lukemiseen: https://world.openbeautyfacts.org/api/v2/product/[viivakoodi].json

Esimerkki: https://world.openbeautyfacts.org/api/v2/product/3600550578041.json

  • API V1- Vanha dokumentaatio saatavilla Wikissä (Tärkeä huomautus: sivu on jaettu alasivuihin READ ja WRITE).
  • API V2 - Kattava, mutta ei OpenAPI-yhteensopiva - Se sisältää käyttötapauksia tuotteen näyttämiseen ja antaa käyttäjille mahdollisuuden lisätä tarvittavat tiedot Nutri-Score-, NOVA-luokituksen, lisäaineiden ja Eco-Score-arvioinnin saamiseksi.
  • API V2 - uusi OpenAPI-dokumentaatio - Työ kesken
  • API V3 - uusi OpenAPI-dokumentaatio - Työ kesken

Voit käyttää API:a tuotannossa, kunhan 1 API-kutsu = 1 oikea viivakoodin lukeminen käyttäjältä. Kaikki yritykset kaapia tietokantaa API:a käyttämällä estetään erittäin todennäköisesti, sillä päivittäiset viennit ovat saatavilla tältä samaiselta sivulta.

Android- ja iPhone-mobiilisovellus (apua tarvitaan!)

Open Beauty Facts -mobiilisovelluksen koodi on saatavilla GitHubista (Android ja iOS). Tarvitsemme apua uuden iOS-koodin mukauttamisessa kosmetiikkaan Sovelluksen avulla käyttäjät voivat skannata tuotteiden viivakoodit, tarkastella tuotetietoja sekä ottaa ja lähettää kuvia ja tietoja puuttuvista tuotteista.

Android-sovellus on kehitetty Kotlinilla.

Uusi cross-platform-sovellus on kehitetty Flutterilla/Dartilla.

Voit uudelleen käyttää koodia (se on avointa lähdekoodia), ja auttaa meitä parantamaan sitä, kaikille, koko planeetalla.
