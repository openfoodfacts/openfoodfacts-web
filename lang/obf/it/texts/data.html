Dati

I singoli contenuti del database sono disponibili sotto la Licenza dei Contenuti del Database.
Le immagini dei prodotti sono disponibili sotto la Licensa Creative Commons Attribution ShareAlike. Potrebbero contenere elementi grafici soggetti a diritto di copia o altri diritti, che potrebbero in alcuni casi essere riprodotti (diritti di quotazione o fair use).

Sei pregato di leggere i Termini e le condizioni di utilizzo e riutilizzo, prima di riutilizzare i dati.

Siamo molto interessati a scoprire per cosa sono utilizzati i dati di Open Beauty Facts. Non è obbligatorio, ma apprezzeremmo molto se ci parlassi dei tuoi riutilizzi, così che possiamo condividerli con la community di Open Beauty Facts.

Esportazione Delta

Le esportazioni delta giornaliere sono fornite per i precedenti 14 giorni. L'elenco di file delta correntemente disponibili si può trovare in https://static.openbeautyfacts.org/data/delta/index.txt. Ogni riga rappresenta un file disponibile in https://static.openbeautyfacts.org/data/delta/{filename}. Il nome del file contiene un timestamp UNIX della prima e dell'ultima modifica contenuta nel file JSON, così che i file delta possano essere importati (dopo l'estrazione) con mongoimport in ordine alfabetico.

API JSON live

L'API si comporta al 99% come su Open Food Facts, quindi molti SDK, i codici per interagire con l'API di Open Food Facts, dovrebbero funzionare semplicemente modificando l'endpoint da: openfoodfacts.org a openbeautyfacts.org. Il codice non è stato aggiornato per oltre un anno, quindi molte API più nuove come Knowledge Panels non sono ancora disponibili.

Un API JSON è anche disponibile per leggere i dati per un prodotto. Questa API, in particolare, è utilizzata nell'app mobile di Open Beauty Facts per iPhone e Android.

Modifichiamo l'API nel tempo, ma stiamo attenti a mantenere la compatibilità con le versioni precedenti. Per favore, dicci se lo usi, così che possiamo dirti di potenziali cambiamenti.

URL per leggere i dati per un prodotto: https://world.openbeautyfacts.org/api/v2/product/[barcode].json

Esempio: https://world.openbeautyfacts.org/api/v2/product/3600550578041.json

  • API V1: Vecchia documentazione disponibile sulla Wiki (Nota importante: la pagina è divisa in sub-pagine per LETTURA e SCRITTURA).
  • API V2: Completa ma non conforme a OpenAPI - Contiene i casi d'uso per la visualizzazione di un prodotto e consente ai tuoi utenti di aggiungere i dati necessari per ottenere il Nutri-Score, la classificazione NOVA, gli additivi e l'Eco-Score.
  • API V2: Nuova documentazione di OpenAPI - Lavori in Corso
  • API V3: Nuova documentazione di OpenAPI - Lavori in Corso

Sei benvenuto all'uso dell'API per i casi di produzione, finché 1 chiamata dell'API equivale ad una scansione reale da un utente. Ogni tentativo di raschiare il database usando l'API sarà molto probabilmente bloccata, poiché le esportazioni giornaliere complete sono disponibili su questa pagina.

Inoltre, inviare un'intestazione HTTP con la tua chiamata dell'API è un buon modo in cui possiamo metterci in contatto, dovesse sorgere qualsiasi problema.

Se sei insicuro di qualcosa, o hai qualche domanda (non ci sono domande stupide), sentiti libero di chiedere su Slack :-)

App mobile per Android e iPhone (aiuto necessario!)

Il codice per l'app mobile di Open Beauty Facts è disponibile su GitHub (Android e iOS). Ci serve aiuto per adattare il nuovo codice nativo di iOS ai cosmetici L'app consente agli utenti di scansionare il codice a barre dei prodotti, di visualizzarne le informazioni e di scattare e inviare foto e dati per i prodotti mancanti.

L'app per Android è sviluppata in Kotlin.

La nuova app multi-piattaforma è sviluppata in Flutter/Dart.

Sei il benvenuto per riutilizzare il codice (è open source) ed aiutarci a migliorarla, per tutti, su tutto il pianeta.
