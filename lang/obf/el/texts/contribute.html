Ζητείται βοήθεια από ειδικούς

Η ανάπτυξη του Open Beauty Facts απαιτεί επίσης εξειδικευμένες γνώσεις και εμπειρία σε πολλούς διαφορετικούς τομείς:

  • Διαχείριση Έργου

    Έχουμε άπειρες ιδέες και εσείς πιθανώς έχετε ακόμη περισσότερες, αλλά είναι αρκετά δύσκολο να τις ιεραρχήσουμε κατά προτεραιότητα, να δημιουργήσουμε έναν οδικό χάρτη και να διαχειριστούμε έργα όταν κάθε συμμετέχων είναι εθελοντής με συχνά περιορισμένο διαθέσιμο χρόνο.

  • Σχεδίαση

    Χρειαζόμαστε βοήθεια για να δημιουργήσουμε μια καλύτερη εμπειρία χρήστη στον ιστότοπο και την εφαρμογή Open Beauty Facts για κινητά, να βελτιώσουμε τον σχεδιασμό τους, να δημιουργήσουμε υλικό παρουσίασης με αντίκτυπο κ.λπ.

  • Ανάπτυξη

    Έχουμε πολλή δουλειά να κάνουμε στην ανάπτυξη: στο backend του Open Beauty Facts (Perl και MongoDB), στο API (JSON), στον ιστότοπο (HTML5, JS, Foundation), στις εφαρμογές iOS και Android (προς το παρόν Apache Cordova / Phonegap, με μια εγγενή έκδοση Android σχεδόν έτοιμη και πολλή δουλειά να κάνουμε στο iOS), για να δημιουργήσουμε νέες ενδιαφέρουσες επαναχρήσεις κ.λπ. Ο κώδικάς μας βρίσκεται στο GitHub (Android, iOS, Cordova.)

  • Δημιουργία κοινοτήτων

    Πρέπει να δημιουργήσουμε τοπικές κοινότητες σε όλες τις χώρες και ταυτόχρονα να τις ενώσουμε σε παγκόσμιο επίπεδο.

    Είναι πολύ δύσκολο να δημιουργήσεις μια τοπική κοινότητα σε μια χώρα χωρίς να ζεις εκεί, επομένως η βοήθειά σας για να βρείτε τους πρώτους πολύ ενθουσιώδεις συμμετέχοντες είναι απαραίτητη. Είστε ένας από αυτούς;

  • Επικοινωνία

    Δεν είμαστε σίγουροι τι να γράψουμε εδώ, θα μπορούσατε να βοηθήσετε;

    Πιο σοβαρά, υπάρχουν πολλά ενδιαφέροντα πράγματα που κάνουν οι χρήστες, οι συνεισφέροντες και οι επαναχρήστες με το Open Beauty Facts, θα ήταν υπέροχο να το μάθουν περισσότεροι άνθρωποι. Συγκεκριμένα, χρειαζόμαστε βοήθεια για τις δημόσιες και τις σχέσεις με τα μέσα ενημέρωσης.

  • Ειδικά έργα

    Υπάρχουν πολλές ενδιαφέρουσες και πρωτότυπες εφαρμογές των ανοιχτών δεδομένων καλλυντικών στις οποίες θα μπορούσαμε να συνεργαστούμε με κυβερνητικούς οργανισμούς καλλυντικών, παραγωγούς καλλυντικών, ερευνητές, πανεπιστήμια, σχολεία, ΜΚΟ κ.λπ. Ίσως έχετε ήδη ιδέες; Αν ναι, παρακαλούμε να μας βοηθήσετε να τους προωθήσουμε.
