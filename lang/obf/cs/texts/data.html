Delta Export

Denní delta exporty jsou poskytovány za předchozích 14 dní. Seznam aktuálně dostupných delta souborů lze nalézt na https://static.openbeautyfacts.org/data/delta/index.txt. Každý řádek představuje soubor, který je k dispozici na https://static.openbeautyfacts.org/data/delta/{filename}. Název souboru obsahuje UNIX časové razítko prvního a poslední změny obsažené v souboru JSON, aby delta soubory mohly být importovány (po extrakci) pomocí mongoimport v abecedním pořadí.

Live JSON API

API se chová z 99 % jako Open Food Facts, takže mnoho SDK, kódů pro interakci s Open Food Facts API by mělo fungovat pouze změnou koncového bodu z: openfoodfacts.org na openbeautyfacts.org. Kód nebyl aktualizován více než rok, takže mnoho novějších rozhraní API, jako jsou panely znalostí, ještě není k dispozici.

K dispozici je také JSON API pro čtení dat pro produkt. Toto API se používá zejména v mobilní aplikaci Open Beauty Facts pro iPhone a Android.

Postupně měníme rozhraní API, ale dbáme na zachování zpětné kompatibility. Řekněte nám, zda ho používáte, abychom vám mohli sdělit o možných změnách.

Adresa URL pro čtení dat pro produkt: https://world.openbeautyfacts.org/api/v2/product/[barcode].json

Příklad: https://world.openbeautyfacts.org/api/v2/product/3600550578041.json

  • API V1- Stará dokumentace dostupná na Wiki (Důležitá poznámka: stránka je rozdělena na podstránky pro ČTENÍ a ZÁPIS).
  • API V2 – Komplexní, ale není kompatibilní s OpenAPI – Obsahuje případy použití pro zobrazení produktu a umožňuje vašim uživatelům přidávat údaje potřebné k získání Nutri-Score, klasifikace NOVA, přísad a Eco-Score.
  • API V2 - nová dokumentace OpenAPI - Work in Progress
  • API V3 - nová dokumentace OpenAPI - Work in Progress

Je velmi vítáno používat API pro produkční případy, pokud 1 volání API = 1 skutečné skenování uživatelem. Jakýkoli pokus o stažení databáze pomocí API bude velmi pravděpodobně zablokován, protože právě na této stránce jsou k dispozici úplné denní exporty.

Také odeslání HTTP hlavičky s vaším voláním API je dobrý způsob, jak se s námi spojit, pokud by nastal nějaký problém.

Pokud si s něčím nevíte rady, nebo máte nějaký dotaz (není to žádná hloupá otázka), klidně se zeptejte na Slacku :-)

Mobilní aplikace pro Android a iPhone (potřebujeme pomoc!)

Kód pro mobilní aplikaci Open Beauty Facts je k dispozici na GitHubu (Android a iOS). Potřebujeme pomoc s přizpůsobením nového nativního iOS kódu kosmetice Aplikace umožňuje uživatelům naskenovat čárový kód produktů, zobrazit informace o produktu a pořizovat a odesílat fotografie a údaje o chybějících produktech.

Aplikace pro Android je vyvinuta v Kotlin.

Nová aplikace pro různé platformy je vyvinuta ve Flutter/Dart.

Můžete znovu použít kód (je to open source) a pomoci nám jej vylepšit, pro každého na celé planetě.
