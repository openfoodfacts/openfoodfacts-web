அறிவியல் வெளியீடுகள்

ஆராய்ச்சியில் திறந்த உணவு உண்மைகள்: வெளியீடுகள், மேற்கோள்கள் மற்றும் ஆராய்ச்சியாளர்களுக்கான வளங்கள்.

2012 முதல், ஊட்டச்சத்து, சுற்றுச்சூழல், பொது சுகாதாரம் மற்றும் பல துறைகள் குறித்த ஆய்வுகளுக்கு எங்கள் உணவுத் தரவைப் பயன்படுத்தும் உலகளாவிய ஆராய்ச்சியாளர்களுக்கு Open Food Facts ஒரு மதிப்புமிக்க வளமாக மாறியுள்ளது. எங்கள் திறந்த தரவுத்தளம் நூற்றுக்கணக்கான அறிவியல் ஆய்வுக் கட்டுரைகளில் மேற்கோள் காட்டப்பட்டுள்ளது.

  • தரவுத்தள ஏற்றுமதிகள்: பல்வேறு வடிவங்களில் முழுமையான தரவுத்தள டம்ப்களைப் பதிவிறக்கவும். விவரங்களுக்கு எங்கள் தரவுப் பக்கத்தைப் பார்க்கவும்.
  • API: எங்கள் விரிவான API வழியாக எங்கள் தரவுத்தளத்தை நிரல் ரீதியாக வினவவும். API ஆவணம்
  • தேடல் இடைமுகம்: மேம்பட்ட தேடல் மற்றும் வடிகட்டுதல் திறன்களுடன் எங்கள் வலைத்தளத்தில் நேரடியாக தயாரிப்புகளை ஆராயுங்கள்.

உரிமம் வழங்குதல்

எங்கள் தரவு திறந்திருக்கும் மற்றும் சுதந்திரமாக மீண்டும் பயன்படுத்தக்கூடியது:

  • இந்தத் தரவுத்தளம் திறந்த தரவுத்தள உரிமம் (ODbL)இன் கீழ் கிடைக்கிறது.
  • தரவுத்தள உள்ளடக்கங்கள் தரவுத்தள உள்ளடக்க உரிமம் (DbCL)இன் கீழ் உள்ளன.
  • தயாரிப்பு படங்கள் கிரியேட்டிவ் காமன்ஸ் அட்ரிபியூஷன் ஷேர்அலைக் (CC BY-SA 3.0)இன் கீழ் கிடைக்கின்றன.

Data Quality

கூட்ட நெரிசல் தரவுத்தளமாக, தரவு தரம் மாறுபடலாம். ஆராய்ச்சிக்காக திறந்த உணவு உண்மைகளைப் பயன்படுத்தும் போது:

  • முழுமை மதிப்பெண்கள் அல்லது தரவு தர குறிகாட்டிகள் மூலம் வடிகட்டுவதைக் கருத்தில் கொள்ளுங்கள்.
  • துல்லியம் அவசியமானால் முக்கியமான தரவுப் புள்ளிகளைச் சரிபார்க்கவும்.
  • எங்கள் தரவு கவரேஜில் புவியியல் மற்றும் தற்காலிக சார்புகள் குறித்து எச்சரிக்கையாக இருங்கள்.
  • உங்கள் ஆராய்ச்சிக்கான தரவு தர மதிப்பீட்டில் உதவி தேவைப்பட்டால் எங்களைத் தொடர்பு கொள்ளவும்.

எங்களுடன் இணைந்து பணியாற்றுங்கள்

  • தரவு தேவைகள்: குறிப்பிட்ட தரவு அல்லது எங்கள் தரவுத்தள கட்டமைப்பில் மேம்பாடுகள் தேவைப்பட்டால் எங்களுக்குத் தெரியப்படுத்துங்கள்
  • ஸ்கேன் பார்ட்டிகள்: உங்கள் ஆராய்ச்சிப் பகுதி அல்லது தயாரிப்பு வகைக்கான இலக்கு தரவு சேகரிப்பு பிரச்சாரங்களை ஒழுங்கமைக்கவும்
  • முடிவுகள் பகிர்வு: உங்கள் கண்டுபிடிப்புகளை எங்கள் சமூகத்துடன் பகிர்ந்து அறிவை மேம்படுத்த உதவுங்கள்
  • கருவி மேம்பாடு: ஆராய்ச்சியாளர்கள் மற்றும் நுகர்வோர் இருவருக்கும் பயனளிக்கும் கருவிகளில் ஒத்துழைக்கவும்

இணைந்திருங்கள்

விஞ்ஞானிகளுக்கான இணையவழிக் கருத்தரங்குகள்

திறந்த உணவு உண்மைகள் மற்றும் ஆராய்ச்சியில் அதன் பயன்பாடுகள் குறித்து விவாதிக்க பிரெஞ்சு மற்றும் ஆங்கில மொழிகளில் ஆராய்ச்சியாளர்களுக்கான வெபினார்கள் நடத்துகிறோம்.

பதிவு (ஆங்கிலம்) பதிவு (பிரெஞ்சு)

திறந்த உணவு உண்மைகள்: 2012 முதல் அறிவியல் சார்ந்த மற்றும் உதவும் அறிவியல்.
