O banco de dados de produtos cosméticos

Open Beauty Facts é uma base de dados de produtos cosméticos com ingredientes, alergénios, vestígios de ingredientes e todos os pedacinhos de informação que se encontram nos rótulos dos produtos.

Feito por todos

A Open Beauty Facts é uma associação sem fins lucrativos de voluntários.

Mais que 1800 contribuintes como você adicionaram 13 000+ produtos de 134 países usando o nosso app Android iPhone ou Windows Phone ou sua câmera para digitalizar códigos de barras e enviar de fotos de produtos e os seus rótulos.

Para todos

Os dados sobre cosmeticos são de interesse público e devem ser abertos. A base de dados completa é publicada como dados abertos e pode ser reutilizada por qualquer um e para qualquer uso. Cheque as utilizações incríveis ou faça sua própria!

Faça melhores escolhas de cosméticos

Conhecimento é poder. Conhecimento é poder.

  • Decodifique rótulos

    Aditivos cosméticos, códigos FIL, números CI, alérgenos, códigos de embalagem... Open Beauty Facts ajuda a entender as letras miúdas dos rótulos dos produtos.

    Também classificamos a qualidade nutricional de cada produto.

  • Encontrar produtos

    O nosso formulário de pesquisa inclui dezenas de critérios para que possa encontrar produtos que correspondam aos seus critérios.

  • Compare e mude

    Você sabe quais champôs têm menos produtos químicos? Em 3 cliques, pode criar um gráfico interativo para mostrá-lo. Assim, pode mudar para champôs mais saudáveis ​​e, comparando produtos de marcas diferentes, incentiva os produtores a torná-los melhores.

  • Explore e descubra

    Curioso sobre comida? Explore produtos por área, como marcas, categorias, rótulos, origens de ingredientes, etc. As áreas podem ser combinadas para que descubra novos conhecimentos, por exemplo, para mapear as origens dos ingredientes que possuem um rótulo de comércio justo.

Cosméticos, para inovação e ciência

Nossos dados podem ser reutilizados para educação, novos produtos e serviços, pesquisas e muito mais!

O banco de dados do Open Beauty Facts são dados abertos publicados sob a Open Database License. Qualquer pessoa pode usá-lo para qualquer finalidade - comercial ou não comercial - desde que a fonte dos dados seja atribuída e as melhorias no banco de dados sejam compartilhadas de maneira semelhante.
