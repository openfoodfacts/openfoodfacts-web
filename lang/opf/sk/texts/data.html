Open Products Facts data exports

Výpisy databázy a exporty sa generujú každú noc.

Informácie o rôznych poliach pre výpis MongoDB a exporty CSV sú dostupné na https://world.openproductsfacts.org/data/data-fields.txt

Výpis MongoDB

Údaje pre všetky produkty sú dostupné vo výpise databázy MongoDB.

Delta Export

Denný export delta sa poskytuje za predchádzajúcich 14 dní. Zoznam aktuálne dostupných delta súborov nájdete na https://static.openproductsfacts.org/data/delta/index.txt. Každý riadok predstavuje súbor, ktorý je dostupný na adrese https://static.openproductsfacts.org/data/delta/{filename}. Názov súboru obsahuje UNIXovú časovú pečiatku prvej a poslednej zmeny obsiahnutej v súbore JSON, takže delta súbory možno importovať (po extrakcii) s mongoimport v abecednom poradí .

Please note that due to the nature of mongoexport, the delta files cannot tell you about deleted products. To remove deleted products from your database, you will need to import the full MongoDB dump.

CSV Data Export

Údaje pre všetky produkty alebo niektoré produkty je možné stiahnuť vo formáte CSV (čitateľné pomocou OpenOffice, Excel a mnohých ďalších softvérov) prostredníctvom formulára pre pokročilé vyhľadávanie.

Odkaz
https://static.openproductsfacts.org/data/en.openproductsfacts.org.products.csv

The file encoding is Unicode UTF-8. The character that separates fields is <tab> (tabulation).

Live JSON API

Rozhranie API sa na 99 % správa ako Open Food Facts, takže mnohé z SDK, kódov na interakciu s Open Food Facts API by mali fungovať jednoducho zmenou koncového bodu z: openfoodfacts.org na openproductsfacts.org. Kód nebol aktualizovaný viac ako rok, takže mnohé z novších rozhraní API, ako sú znalostné panely, ešte nie sú k dispozícii.

A JSON API is also available to read the data for a product. Toto API sa používa najmä v mobilnej aplikácii Open Products Facts pre iPhone a Android.

Rozhranie API v priebehu času meníme, ale dávame pozor, aby sme zachovali spätnú kompatibilitu. Povedzte nám, či ho používate, aby sme vás mohli informovať o možných zmenách.

Webová adresa na čítanie údajov o produkte: https://world.openproductsfacts.org/api/v2/product/[čiarový kód].json

Príklad: https://world.openproductsfacts.org/api/v2/product/8710847992063.json

  • API V1- Stará dokumentácia dostupná na Wiki (Dôležitá poznámka: stránka je rozdelená na podstránky pre ČÍTANIE a ZÁPIS).
  • API V2 – Komplexné, ale nie v súlade s OpenAPI – Obsahuje prípady použitia na zobrazenie produktu a umožňuje vašim používateľom pridať údaje potrebné na získanie Nutri-Score, klasifikácie NOVA, aditív a ekologické skóre.
  • API V2 - nová dokumentácia OpenAPI - Work in Progress
  • API V3 - nová dokumentácia OpenAPI - Work in Progress

You are very welcome to use the API for production cases, as long as 1 API call = 1 real scan by a user. Any attempt to scrape the database using the API will very likely be blocked, as full daily exports are available on this very page.

Also, sending an HTTP header with your API call is a good way we can get in touch, should any issue arise.

Ak si niečím nie ste istí, alebo máte akúkoľvek otázku (nie je to hlúpa otázka), pokojne sa opýtajte na Slack :-)

Mobilná aplikácia pre Android a iPhone (potrebná pomoc!)

Kód aktuálne nasadenej mobilnej aplikácie Open Products Facts je k dispozícii na GitHub (Android. Potrebujeme pomoc s prispôsobením nového kódu Flutter kozmetike Aplikácia umožňuje používateľom naskenovať čiarový kód produktov a zobraziť informácie o produkte a na nasnímanie a odoslanie obrázkov a údajov o chýbajúcich produktoch.

Aplikácia pre Android je vyvinutá v Kotlin.

Nová multiplatformová aplikácia je vyvinutá vo Flutter/Dart.

You are welcome to reuse the code (it's open source) and help us improve it, for everyone, on all the planet.
