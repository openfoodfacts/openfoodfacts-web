• Description : Ce descripteur indique que le jeu contient un langage offensant ou des jurons.
  • Niveaux de gravité :
  • Léger (PEGI 12) : Jurons légers et/ou langage offensant, sans jurons sexuels (par exemple, « zut », « enfer », « merde »).
  • Fort (PEGI 16/18) : Jurons sexuels, blasphèmes ou langage très offensant.
  • Implication : La présence et l’intensité du langage grossier augmentent avec l’âge.
  • Sources : Site Web officiel PEGI, Autorité de classification des jeux
