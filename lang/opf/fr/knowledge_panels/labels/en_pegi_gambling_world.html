• De quoi s'agit-il : Ce descripteur indique que le jeu contient des éléments qui encouragent ou enseignent le jeu.
  • Portée : Fait référence aux simulations de jeux de hasard traditionnels que l'on trouve généralement dans les casinos ou les salles de jeux (par exemple, les jeux de cartes joués pour de l'argent, les paris sur les courses).
  • Classification minimale : Ce descripteur n'apparaît que sur les jeux classés PEGI 12, 16 ou 18. Depuis 2020, les jeux nouvellement classés avec des simulations de jeu réalistes sont toujours PEGI 18.
  • Remarque sur les boîtes à butin : La présence d'« objets aléatoires payants » (souvent appelés boîtes à butin) est un descripteur de contenu distinct (« Achats en jeu ») et peut être trouvée dans les jeux avec des classifications d'âge inférieures (par exemple, PEGI 7).
