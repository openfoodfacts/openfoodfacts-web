• De quoi s'agit-il : Ce descripteur indique que le jeu propose des achats numériques optionnels avec de l'argent réel.
  • Portée : Inclut diverses formes d'achats dans le jeu :
  • Objets cosmétiques : Skins, emotes, améliorations visuelles.
  • Avantages de jeu : Power-ups, monnaie, boosters.
  • Extensions/DLC : Contenu ou histoire supplémentaire.
  • Objets aléatoires payants (boîtes à butin) : Lorsque les joueurs ne savent pas exactement ce qu'ils obtiennent avant l'achat (par exemple, packs de cartes, caisses d'objets).
  • Remarque : Ce descripteur n'est pas une classification par âge, mais une icône informative. Il peut apparaître dans des jeux avec n'importe quelle classification d'âge PEGI, y compris PEGI 3.
  • Préoccupations (en particulier pour les enfants) : Potentiel de dépenses excessives, encouragement de comportements de type jeu avec des objets aléatoires et pression d'achat pour progresser.
  • Sources : Site Web officiel PEGI, Zone Parent - Comprendre les classifications PEGI
