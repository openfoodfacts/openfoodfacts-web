• De quoi s'agit-il : Le contenu est considéré comme adapté à tous les groupes d'âge.
  • Caractéristiques clés :
  • Violence : Seules les formes de violence très légères sont acceptables, généralement dans un cadre comique, caricatural ou enfantin (par exemple, burlesque, préjudice implicite, réactions irréalistes).
  • Peur : Pas de sons ni d'images susceptibles d'effrayer les jeunes enfants.
  • Langage : Aucun langage grossier ne doit être entendu.
  • Autre : Pas de contenu sexuel, de drogue ou de jeu.
  • Implication : Cette classification vise à garantir que le contenu est sûr et non dérangeant pour le plus jeune public.
  • Sources : Site Web officiel PEGI, Uswitch - Guide des parents sur les classifications PEGI
