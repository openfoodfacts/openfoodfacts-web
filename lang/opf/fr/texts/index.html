Découvrir

Open Products Facts est une base de données générale sur les produits (aliments, cosmétiques et aliments pour animaux de compagnie exemptés) créée par tout le monde, pour tout le monde. Vous pouvez l'utiliser pour faire de meilleurs choix de produits, et comme il s'agit de données libres, tout le monde peut les réutiliser à n'importe quelle fin.
