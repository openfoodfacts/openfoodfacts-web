Descubra

Open Products Facts é uma base de dados de produtos gerais (com isenção de alimentos, cosméticos e pet food) feita por todo mundo, para todo mundo. Você pode usá-lo para fazer melhores escolhas e, como são dados abertos, qualquer pessoa pode reutilizá-los para qualquer finalidade.
