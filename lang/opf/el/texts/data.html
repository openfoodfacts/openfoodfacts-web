Δεδομένα

Όροι επαναχρησιμοποίησης

Η βάση δεδομένων Open Products Facts είναι διαθέσιμη με την Άδεια Ανοικτής Βάσης Δεδομένων.
Τα επιμέρους περιεχόμενα της βάσης δεδομένων είναι διαθέσιμα με την Άδεια Περιεχομένων Βάσης Δεδομένων.
Οι εικόνες προϊόντων είναι διαθέσιμες με την άδεια Creative Commons Attribution ShareAlike. Μπορεί να περιέχουν γραφικά στοιχεία που υπόκεινται σε πνευματικά δικαιώματα ή άλλα δικαιώματα, τα οποία ενδέχεται σε ορισμένες περιπτώσεις να αναπαραχθούν (δικαιώματα προσφοράς ή ορθή χρήση).

Παρακαλούμε διαβάστε τους Όρους και προϋποθέσεις χρήσης και επαναχρησιμοποίησης πριν από την επαναχρησιμοποίηση των δεδομένων.

Πείτε μας για την επαναχρησιμοποίησή σας

Μας ενδιαφέρει πολύ να μάθουμε για ποιο σκοπό χρησιμοποιούνται τα δεδομένα των Ανοικτών Προϊόντων. Δεν είναι υποχρεωτικό, αλλά θα το εκτιμούσαμε ιδιαίτερα αν μας ενημερώσετε για τις επαναχρήσεις σας ώστε να μπορέσουμε να τις κοινοποιήσουμε στην κοινότητα Open Products Facts. Μπορείτε επίσης να συμπληρώσετε αυτήν τη φόρμα για να έχετε την ευκαιρία να αναδείξετε την εφαρμογή σας μέσω εμάς.

Εξαγωγές δεδομένων Open Products Facts

Οι καταχωρήσεις βάσεων δεδομένων και οι εξαγωγές δημιουργούνται κάθε νύχτα. Η βάση δεδομένων είναι πολύ μεγάλη, επομένως μπορείτε να βρείτε διαφορετικούς τρόπους χρήσης των δεδομένων, ανάλογα με τον τρόπο εξαγωγής.

Πληροφορίες σχετικά με τα διάφορα πεδία για το dump MongoDB και τις εξαγωγές CSV είναι διαθέσιμες στη διεύθυνση https://world-opf.openproductsfacts.org/data/data-fields.txt

Καταχώρηση MongoDB

Τα δεδομένα για όλα τα προϊόντα είναι διαθέσιμα σε μια καταχώρηση βάσης δεδομένων MongoDB.

Σύνδεσμος
https://static.openproductsfacts.org/data/openproductsfacts-mongodbdump.gz
sha256sum
https://static.openproductsfacts.org/data/gz-sha256sum
md5sum
https://static.openproductsfacts.org/data/gz-md5sum

Δέλτα Εξαγωγή

Παρέχονται ημερήσιες εξαγωγές δέλτα για τις προηγούμενες 14 ημέρες. Η λίστα με τα αρχεία delta που είναι διαθέσιμα αυτήν τη στιγμή βρίσκεται στη διεύθυνση https://static.openproductsfacts.org/data/delta/index.txt.

Κάθε γραμμή αντιπροσωπεύει ένα αρχείο που είναι διαθέσιμο στη διεύθυνση https://static.openproductsfacts.org/data/delta/{filename}.

Το όνομα αρχείου περιέχει χρονική σήμανση UNIX της πρώτης και της τελευταίας αλλαγής που περιέχεται στο αρχείο JSON, έτσι ώστε τα αρχεία δέλτα να μπορούν να εισαχθούν (μετά την εξαγωγή) με mongoimport σε αλφαβητική σειρά.

Λάβετε υπόψη ότι λόγω της φύσης του mongoexport, τα αρχεία δέλτα δεν μπορούν να σας ενημερώσουν για τα διαγραμμένα προϊόντα. Για να αφαιρέσετε τα διαγραμμένα προϊόντα από τη βάση δεδομένων σας, θα χρειαστεί να εισαγάγετε την πλήρη καταχώρηση MongoDB.

Εξαγωγή δεδομένων JSONL

Ολόκληρη η βάση δεδομένων είναι επίσης διαθέσιμη σε μορφή JSONL (μερικές φορές ονομάζεται LDJSON ή NDJSON) όπου κάθε γραμμή είναι ένα αντικείμενο JSON. Αντιπροσωπεύει τα ίδια δεδομένα με την εξαγωγή σε MongoDB. Το αρχείο είναι συμπιεσμένο με gzip.

Σύνδεσμος
https://static.openproductsfacts.org/data/openproductsfacts-products.jsonl.gz

Ένας κατάλληλος τρόπος για την εκμετάλλευση της βάσης δεδομένων είναι η χρήση του DuckDB, ενός αναλυτικού εργαλείου που έχει σχεδιαστεί για την επεξεργασία μεγάλου όγκου δεδομένων σε κλάσματα δευτερολέπτων. Μπορείτε να διαβάσετε την ανάρτηση ιστολογίου όπου σας καθοδηγούμε στην εξερεύνηση και την επεξεργασία της βάσης δεδομένων Open Products Facts με το DuckDB

Εξαγωγή δεδομένων CSV

Τα δεδομένα για όλα τα προϊόντα ή για ορισμένα από αυτά μπορούν να ληφθούν σε μορφή CSV (αναγνώσιμο με LibreOffice, Excel και πολλά άλλα λογισμικά υπολογιστικών φύλλων) μέσω της φόρμας σύνθετης αναζήτησης .

Σύνδεσμοι
https://static.openproductsfacts.org/data/en.openproductsfacts.org.products.csv.gz (συμπιεσμένο CSV σε μορφή GZIP: ~ 0,9 Gb, μη συμπιεσμένο: ~ 9 Gb)

Η κωδικοποίηση του αρχείου είναι Unicode UTF-8. Ο χαρακτήρας που χωρίζει τα πεδία είναι <tab> (tabulation).

Εξαγωγή Δεδομένων Εικόνας

Όλες οι εικόνες και τα αποτελέσματα OCR μπορούν είτε να ληφθούν από τον διακομιστή μας είτε από το AWS (μέσω του προγράμματος Open Data AWS). Δείτε την τεκμηρίωσή μας για να μάθετε περισσότερα σχετικά με τον τρόπο λήψης εικόνων.

Ζωντανό JSON και XML API

Ένα JSON API είναι επίσης διαθέσιμο για την ανάγνωση των δεδομένων για ένα προϊόν. Αυτό το API χρησιμοποιείται ειδικότερα στην εφαρμογή Open Products Facts για κινητά για iPhone και Android.

Βελτιώνουμε το API με την πάροδο του χρόνου, αλλά προσέχουμε να διατηρήσουμε τη προς τα πίσω συμβατότητα. Παρακαλούμε επικοινωνήστε μαζί μας εάν το χρησιμοποιείτε, ώστε να σας ενημερώσουμε για πιθανές αλλαγές.

If you are unsure about anything, or have any question (there is no silly question), feel free to ask it on Slack :-)

Παράδειγμα: https://world-opf.openproductsfacts.org/api/v3/product/737628064502.json

Σημειώστε ότι εάν η έξοδος είναι υπερβολικά μεγάλη, μπορείτε να τη φιλτράρετε - όπως περιγράφεται στην τεκμηρίωση του API.

  • API V3 - νέα τεκμηρίωση OpenAPI - Εργασία σε εξέλιξη

Ζητείται βοήθεια για το OpenAPI

Ανυπομονούμε να ολοκληρώσουμε την τεκμηρίωση του API, να προσθέσουμε περιπτώσεις χρήσης και να αυτοματοποιήσουμε τη δημιουργία SDK. Εάν είστε γνώστης του OpenAPI, είστε ευπρόσδεκτοι να βοηθήσετε.

Είστε ευπρόσδεκτοι να χρησιμοποιήσετε το API για περιπτώσεις δημόσιας χρήσης, αρκεί 1 κλήση API = 1 πραγματική σάρωση από έναν χρήστη. Οποιαδήποτε προσπάθεια αντιγραφής της βάσης δεδομένων χρησιμοποιώντας το API είναι πολύ πιθανό να αποκλειστεί, καθώς πλήρεις καθημερινές εξαγωγές είναι διαθέσιμες σε αυτήν ακριβώς τη σελίδα.

Επίσης, η αποστολή μιας κεφαλίδας HTTP με την κλήση API σας είναι ένας καλός τρόπος για να έρθουμε σε επαφή, σε περίπτωση που προκύψει οποιοδήποτε πρόβλημα.

Εάν δεν είστε σίγουροι για κάτι ή έχετε οποιαδήποτε ερώτηση (δεν υπάρχουν χαζές ερωτήσεις), μη διστάσετε να τη ρωτήσετε στο κανάλι #api στο Slack :-)

Περιτύλιξη και SDK

Πιθανότατα έχουμε ένα περιτυλιχτή για την αγαπημένη σας γλώσσα προγραμματισμού. Εάν ναι, μπορείτε να το χρησιμοποιήσετε και να το βελτιώσετε. Εάν όχι, μπορείτε να βοηθήσετε στη δημιουργία του.

Θα σας επιτρέψουν να καταναλώνετε δεδομένα, καθώς και να επιτρέπουν στους χρήστες σας να συνεισφέρουν νέα δεδομένα

Γενικές αρχές για να κάνετε τους χρήστες σας χαρούμενους

  • Μπορείτε να αναζητήσετε πληροφορίες σχετικά με προϊόντα, συμπεριλαμβανομένων πολλών χρήσιμων υπολογιστικών τιμών.
  • DART: GitHub (Open Food Facts only at the moment)
  • Μπορείτε επίσης να εφαρμόσετε την πλήρη ροή ώστε να πάρουν αμέσως το αποτέλεσμα με μερική προσπάθεια από την πλευρά τους.

Εάν οι χρήστες σας δεν αναμένουν άμεσα αποτέλεσμα (π.χ. εφαρμογές για απόθεμα)

  • Υποβάλετε φωτογραφίες (μπροστά/διατροφή/συστατικά): το πιο ανώδυνο πράγμα για τους χρήστες σας
  • Η Τεχνητή Νοημοσύνη για τα Γεγονότα Ανοικτών Προϊόντων - Robotoff - θα δημιουργήσει ορισμένα παράγωγα δεδομένα από τις φωτογραφίες
  • Με την πάροδο του χρόνου, άλλες εφαρμογές και η κοινότητα Open Products Facts θα καλύψουν τα κενά δεδομένων.

Εάν οι χρήστες σας αναμένουν άμεσα αποτελέσματα (π.χ. εφαρμογές διατροφής ή αλλεργιών)

  • Υποβολή συστατικών + κατηγορία > λήψη ανάλυσης συστατικών, πρόσθετα, αλλεργιογόνα, κανονικοποιημένα συστατικά, vegan, χορτοφαγικό…
  • Υποβολή ετικετών για προστιθέμενη αξία
  • Υποβολή διατροφής

Εφαρμογή για κινητά Android και iPhone

Ο κώδικας για την εφαρμογή Open Products Facts για κινητά είναι διαθέσιμος στο GitHub (Flutter, Kotlin Android και Swift iOS).

Προσπαθούμε να μετατρέψουμε τις καταργημένες βάσεις κώδικα Kotlin και Swift σε Kotlin και Swift SDKs, κάθε βοήθεια ευπρόσδεκτη :-) Η εφαρμογή επιτρέπει στους χρήστες να σαρώνουν τον γραμμωτό κώδικα των προϊόντων, να προβάλλουν τις πληροφορίες του προϊόντος και να τραβούν και να υποβάλλουν φωτογραφίες και δεδομένα για προϊόντα που λείπουν.

Η εφαρμογή επιτρέπει στους χρήστες να σαρώνουν τον γραμμωτό κώδικα των προϊόντων, να προβάλλουν τις πληροφορίες του προϊόντος και να λαμβάνουν και να υποβάλλουν φωτογραφίες και δεδομένα για προϊόντα που λείπουν.

Μπορείτε να χρησιμοποιήσετε ξανά τον κώδικα (είναι ανοιχτού κώδικα) και να μας βοηθήσετε να τον βελτιώσουμε, για όλους, σε όλο τον πλανήτη.
