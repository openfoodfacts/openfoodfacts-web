Όλοι μπορούν να συνεισφέρουν

Γίνετε μέρος της συνεργατικής, δωρεάν και ανοιχτής βάσης δεδομένων μας με προϊόντα από όλο τον κόσμο!

Το Open Products Facts είναι ένα μη κερδοσκοπικό έργο που δημιουργήθηκε εξ ολοκλήρου από εθελοντές, σας χρειαζόμαστε.

Προσθέστε προϊόντα

Χρησιμοποιήστε την εφαρμογή Open Food Facts για Android ή iPhone για να σαρώσετε εύκολα τον γραμμωτό κώδικα προϊόντων από το σπίτι σας ή από τα τοπικά καταστήματα και να ανεβάσετε φωτογραφίες των ετικετών τους.

Οι εφαρμογές για κινητά έχουν συγχωνευτεί, επομένως μπορείτε πλέον να σαρώνετε προϊόντα, τρόφιμα, καλλυντικά και τροφές για κατοικίδια από την ίδια εφαρμογή Open Food Facts!

Δεν έχετε smartphone; Κανένα πρόβλημα: μπορείτε επίσης να χρησιμοποιήσετε την κάμερά σας για να προσθέσετε προϊόντα απευθείας στον ιστότοπο.

Εγκαταστήστε την εφαρμογή για να συμβάλετε στη διαφάνεια του προϊόντος

Πείτε στον κόσμο

Σας αρέσουν τα Open Products Facts; Πείτε σε άλλους για αυτό!

Μπορείτε να παρουσιάσετε το έργο στην οικογένεια και τους φίλους σας, να τους δείξετε πώς να εγκαταστήσουν την εφαρμογή και να συνεισφέρετε, να γράψετε μια ανάρτηση ιστολογίου, να μοιραστείτε πληροφορίες για τα ανοιχτά προϊόντα στα μέσα κοινωνικής δικτύωσης...

Και να το παρουσιάσουν ζωντανά! Έχουμε παρουσιάσει τα Γεγονότα για τα Ανοικτά Προϊόντα σε πολύ διαφορετικά κοινά σε πολύ διαφορετικά περιβάλλοντα και λαμβάνουμε πάντα μια πολύ ενθουσιώδη ανταπόκριση όταν δείχνουμε τι μπορεί να γίνει όταν τα δεδομένα για τα προϊόντα είναι ανοιχτά. Διαθέτουμε πολλά από υλικά παρουσίασης (διαφάνειες κ.λπ.) που μπορείτε να προσαρμόσετε ή να παρουσιάσετε ως έχουν.

Κάντε το τοπικό

Μπορείτε να βοηθήσετε στη μετάφραση του ιστότοπου και της εφαρμογής για κινητά στη γλώσσα σας, και στη μετάφραση παρουσιάσεων, ανακοινώσεων κ.λπ.

Μπορούμε να συνεργαστούμε για να υποστηρίξουμε και να προσθέσουμε τοπικές περιβαλλοντικές ετικέτες, βαθμολογίες επισκευασιμότητας και πρότυπα προϊόντων.

Ξεκινήστε ή γίνετε μέλος μιας τοπικής κοινότητας συνεργατών: προσθέστε τοπικά προϊόντα, προσελκύστε φίλους, παρουσιάστε το έργο σε τοπικές συναντήσεις και συνέδρια κ.λπ.

Ζητείται βοήθεια από ειδικούς

Η ανάπτυξη Γεγονότων για τα Ανοικτά Προϊόντα απαιτεί επίσης εξειδικευμένες γνώσεις και εμπειρογνωμοσύνη σε πολλούς διαφορετικούς τομείς:

  • Διαχείριση Έργου

    Έχουμε άπειρες ιδέες και εσείς πιθανώς έχετε ακόμη περισσότερες, αλλά είναι αρκετά δύσκολο να τις ιεραρχήσουμε κατά προτεραιότητα, να δημιουργήσουμε έναν οδικό χάρτη και να διαχειριστούμε έργα όταν κάθε συμμετέχων είναι εθελοντής με συχνά περιορισμένο διαθέσιμο χρόνο.

  • Σχεδίαση

    Χρειαζόμαστε βοήθεια για να δημιουργήσουμε μια καλύτερη εμπειρία χρήστη στον ιστότοπο και την εφαρμογή για κινητά του Open Products Facts, να βελτιώσουμε τον σχεδιασμό τους, να δημιουργήσουμε υλικό παρουσίασης με αντίκτυπο κ.λπ.

  • Ανάπτυξη

    Έχουμε πολλή δουλειά να κάνουμε στην ανάπτυξη: στο backend του Open Products Facts (Perl και MongoDB), στο API (JSON), στον ιστότοπο (HTML5, JS, Foundation) και στην εφαρμογή για κινητά. Ο κώδικάς μας είναι στο GitHub.

  • Δημιουργία κοινοτήτων

    Πρέπει να δημιουργήσουμε τοπικές κοινότητες σε όλες τις χώρες και ταυτόχρονα να τις ενώσουμε σε παγκόσμιο επίπεδο.

    Είναι πολύ δύσκολο να δημιουργήσεις μια τοπική κοινότητα σε μια χώρα χωρίς να ζεις εκεί, επομένως η βοήθειά σας για να βρείτε τους πρώτους πολύ ενθουσιώδεις συμμετέχοντες είναι απαραίτητη. Είστε ένας από αυτούς;

  • Επικοινωνία

    Δεν είμαστε σίγουροι τι να γράψουμε εδώ, θα μπορούσατε να βοηθήσετε;

    Πιο σοβαρά, υπάρχουν πολλά ενδιαφέροντα πράγματα που κάνουν οι χρήστες, οι συνεισφέροντες και οι επαναχρήστες με το Open Products Facts, θα ήταν υπέροχο να το μάθουν περισσότεροι άνθρωποι. Συγκεκριμένα, χρειαζόμαστε βοήθεια για τις δημόσιες και τις σχέσεις με τα μέσα ενημέρωσης.

  • Ειδικά έργα

    Υπάρχουν πολλές ενδιαφέρουσες και πρωτότυπες εφαρμογές των ανοιχτών δεδομένων προϊόντων στις οποίες θα μπορούσαμε να συνεργαστούμε με περιβαλλοντικούς φορείς, κατασκευαστές, ερευνητές, πανεπιστήμια, σχολεία, ΜΚΟ κ.λπ. Ίσως έχετε ήδη ιδέες; Αν ναι, παρακαλούμε να μας βοηθήσετε να τους προωθήσουμε.

  • Wiki

    Έχουμε επίσης ένα wiki που χρησιμοποιούμε για τη συλλογική τεκμηρίωση Γεγονότων Ανοικτών Προϊόντων και των υποέργων τους. Χρησιμοποιούμε επίσης το wiki για να διαχειριζόμαστε τις μεταφράσεις και για να επεξεργαστούμε τις ταξινομήσεις / ιεραρχίες κατηγοριών, ετικετών κ.λπ.

Υπάρχουν πολλοί περισσότεροι τρόποι για να συνεισφέρετε στο Open Products Facts, παρακαλούμε ελάτε μαζί μας στο Slack και ας ξεκινήσουμε τη συζήτηση!
