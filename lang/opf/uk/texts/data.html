Експорт дельти

Щоденний експорт дельти представлені на попередні 14 днів. The list of currently available delta files can be found at https://static.openproductsfacts.org/data/delta/index.txt. Each line represents a file that is available at https://static.openproductsfacts.org/data/delta/{filename}. Назва файлу містить часову позначку UNIX на першій та останній зміні, що міститься у файлі JSON, таким чином дельта файли можуть бути імпортовані (після витягнення) з mongoimport у алфебетному порядку.

Експорт даних CSV

Data for all products, or some of the products, can be downloaded in CSV format (readable with OpenOffice, Excel and many other software) through the advanced search form.

Силка
https://static.openproductsfacts.org/data/en.openproductsfacts.org.products.csv

Кодування файлу є юнікодом UTF-8. Символ, що розділяє поля є <tab> (табуляція).

Діючий API JSON

The API behaves 99% like Open Food Facts, so many of the SDKs, codes to interact with the Open Food Facts API should work by merely changing the endpoint from: openfoodfacts.org to openproductsfacts.org. The code hasn't been updated in over a year, so many of the newer APIs like Knowledge Panels are not available yet.

API JSON також є наявним для читання даних продукту. This API is in particular used in the Open Products Facts mobile app for iPhone and Android.

Ми з часом змінюємо API, проте ми ставимося з обережністю до підтримання зворотньої сумісності. Будь ласка розкажіть нам чи ви використовуєте це, таким чином ми зможемо розповісти вам про наші потенційні зміни.

URL to read data for a product: https://world.openproductsfacts.org/api/v2/product/[barcode].json

Example: https://world.openproductsfacts.org/api/v2/product/8710847992063.json

  • API V1 - Стара документація доступна на Вікі (Важливе зауваження: сторінка розділена на підсторінки для ЧИТАННЯ і ЗАПИСУ).
  • API V2 - повний, але не сумісний з OpenAPI - містить варіанти використання для відображення продукту і дозволяє вашим користувачам додавати дані, необхідні для отримання Nutri-Score, класифікації NOVA, добавок та Eco-Score.
  • API V2 - нова документація OpenAPI - Work in progress
  • API V3 - нова документація по OpenAPI - Робота в процесі

Ми дуже охочі до використання API для виробничих кейсів, доки 1 API = 1 справжньому скануванню користувача. Будь-яка спроба здерти базу даних використовуючи API буде ймовірно заблокованою, так як повний щоденний експорт саме на цій сторінці.

Також, надсилання HTTP заголовку з вашим викликом API є хорошим способом для нашого зв'язку, якщо виникне будь-яке питання.

Якщо ви непевні щодо чогось, або маєте якісь запитання (немає дурних запитань), вільно почувайтеся спитати їх у Slack :-)
