Ekspor Delta

Ekspor delta harian disediakan selama periode 14 hari sebelumnya. The list of currently available delta files can be found at https://static.openproductsfacts.org/data/delta/index.txt. Each line represents a file that is available at https://static.openproductsfacts.org/data/delta/{filename}. Nama berkas berisi stempel waktu UNIX mengenai perubahan pertama dan terakhir yang terdapat dalam berkas JSON, sehingga berkas delta dapat diimpor (setelah ekstrasi) dengan mongoimport dalam urutan abjad.

Harap dicatat bahwa karena sifat mongoexport, berkas delta tidak dapat memberi tahu Anda tentang produk - produk yang dihapus. Untuk menghilangkan produk - produk yang dihapus dari basis data Anda, Anda akan harus mengimpor dump MongoDB lengkap.

Ekspor Data CSV

Data for all products, or some of the products, can be downloaded in CSV format (readable with OpenOffice, Excel and many other software) through the advanced search form.

Tautan
https://static.openproductsfacts.org/data/en.openproductsfacts.org.products.csv

Pengodean berkas berbasis Unicode UTF-8. Karakter yang memisahkan field adalah <tab> (tabulation).

API JSON langsung

The API behaves 99% like Open Food Facts, so many of the SDKs, codes to interact with the Open Food Facts API should work by merely changing the endpoint from: openfoodfacts.org to openproductsfacts.org. The code hasn't been updated in over a year, so many of the newer APIs like Knowledge Panels are not available yet.

API JSON juga tersedia untuk membaca data suatu produk. This API is in particular used in the Open Products Facts mobile app for iPhone and Android.

Kami mengubah API dari waktu ke waktu, tapi kami berhati - hati untuk menjaga kompatibilitas ke belakangnya. Harap memberitahu kami jika Anda menggunakannya, sehingga kami dapat memberi tahu Anda tentang kemungkinan perubahannya.

URL to read data for a product: https://world.openproductsfacts.org/api/v0/product/[barcode].json

Example: https://world.openproductsfacts.org/api/v0/product/8710847992063.json

  • API V1- Dokumentasi lama tersedia di Wiki (Catatan penting: halaman ini dibagi menjadi subhalaman untuk BACA dan TULIS).
  • API V2 - Komprehensif tetapi tidak sesuai dengan OpenAPI - Berisi kasus penggunaan untuk menampilkan produk, dan memungkinkan pengguna Anda menambahkan data yang diperlukan untuk mendapatkan Nutri-Score, klasifikasi NOVA, zat tambahan, dan Eco-Score.
  • API V2 - dokumentasi OpenAPI baru - Sedang dalam Proses
  • API V3 - dokumentasi OpenAPI baru - Sedang dalam Proses

Anda dipersilahkan untuk menggunakan API untuk kasus - kasus produksi, selama 1 pemanggilan API = 1 pemindaian nyata oleh pengguna. Upaya apapun untuk mengikis basis data menggunakan API akan sangat mungkin diblokir, karena ekspor harian lengkap tersedia di halaman ini.

Juga, mengirim header HTTP dengan pemanggilan API Anda adalah cara yang baik untuk menghubungi kami, jika ada masalah yang muncul.

Jika Anda tidak yakin tentang apapun, atau memiliki pertanyaan (tidak ada pertanyaan konyol), jangan sungkan untuk menanyakannya di Slack :-)
