Descobrir

O Open Products Facts é uma base de dados de produtos gerais (excepto alimentos, cosméticos e alimentos para animais) feito por todos, para todos. Pode usá-lo para fazer escolhas melhores sobre produtos, e como são dados públicos, qualquer um pode reutilizá-lo para qualquer finalidade.

Contribuir

- O Open Product Facts é um projeto sem fins lucrativos desenvolvido por milhares de voluntários de todo o mundo. Pode começar a contribuir adicionando qualquer tipo de produto com a nossa aplicação para iPhone ou Android, e temos muitos projetos interessantes para os quais pode contribuir de várias maneiras diferentes.
