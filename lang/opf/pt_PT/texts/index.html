Descobrir

O Open Products Facts é uma base de dados de produtos gerais (excepto alimentos, cosméticos e alimentos para animais) feito por todos, para todos. Pode usá-lo para fazer escolhas melhores sobre produtos, e como são dados públicos, qualquer um pode reutilizá-lo para qualquer finalidade.
