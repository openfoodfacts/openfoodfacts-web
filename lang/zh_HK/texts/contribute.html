每个人都可以贡献

成为我们协作、自由、开放的世界食品数据库的一份子！

Open Food Facts 是一个完全由志愿者经营的非营利性项目，我们十分需要你。

添加产品

使用我们 Android, iPhone 或者 Windows Phone 的 app 来扫描家中或者商店里产品的条形码，并且上传它们的标签。

没有智能手机？ 没问题：您也可以使用相机直接在网站上添加产品。

告诉世界

我们需要对产品进行分类，并提取成分表和营养成分来分析它们的营养质量。这可以确定它们是否适合素食者等人群！

You can present the project to your family and friends, show them how to install the app and contribute, write a blog post, share Open Beauty Facts on social media...

告诉世界

你喜欢Open Beauty Facts吗？ 告诉其他人！

我们可以共同协作支持并添加本地标签的徽标，解码本地包装代码以便于我们映射产品等。

将其本地化

You can help to translate the site and mobile app in your language, and to translate presentations, announcements etc.

Start or join a local contributors community: add local products, recruit friends, present the project in local meetups and conferences etc.

需要专家帮助

Developing Open Beauty Facts also requires specialized knowledge and expertise in many different areas:

  • 项目管理

    我们有大量的想法，你可能还有更多的想法，但是要优先考虑它们，制定一个路线图，并在每个参与者通常都只是一个由有限时间的志愿者时管理项目，是一个相当大的挑战。

  • 设计

    We need help to build a better user experience on the Open Beauty Facts web site and mobile app, to improve their design, to create impactful presentation materials etc.

  • 开发

    我们的代码在 GitHub 上。 We have a lot of development work to do: on the Open Beauty Facts backend (Perl and MongoDB), API (JSON), web site (HTML5, JS, Foundation), iOS / Android / Windows Phone / FirefoxOS apps (currently developed in HTML and JS with Apache Cordova / Phonegap), to build new cool reuses etc. Our code is on GitHub. 同时，我们利用人工智能在简化工作方面取得长足的进展， 我们的代码在 GitHub 上。

  • 社区建设

    我们需要在所有国家建立当地社区，同时在全球范围内将它们联合起来。

    It is very difficult to bootstrap a local community in a country without living there, so your help to find the first very motivated participants is essential. 你也是其中之一吗？ 你也是其中之一吗？

  • 通讯

    我们不确定在这里写什么，你能帮忙吗？

    More seriously, there are lot of cool things that users, contributors and reusers do with Open Beauty Facts, it would be great to get more people to know about it. In particular, we need help for public and media relations. 特别是，我们需要公众的帮助，还没媒体关系<0>

  • 特别项目

    There are lots of interesting and original applications of cosmetic open data that we could work on with government cosmetic agencies, cosmetic producers, researchers, universities, schools, NGOs etc. 也许你已经有想法了？ 也许你已经有想法了？ 如果你想这样做，请帮助我们推动他们前进。
