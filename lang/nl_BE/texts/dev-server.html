Draag bij aan de Core met Perl

Wil je een nieuwe taal leren? De server is voornamelijk gecodeerd in Perl. Of je nu een doorgewinterde Perl-verslaafde bent, of net begint met Perl voor sys-admin-taken, er is een bug op jouw maat en mentoren om je te begeleiden. En er zijn ook meerdere manieren om het te doen ;-)

Maak handige tools in elke taal

HungerGames is een voorbeeld van een kleine game die is gecodeerd in React en die een positieve bijdrage heeft aan Open Food Facts. LanguageSwitcher, een andere tool, is gecodeerd in Python. Jouw hulpmiddel, jouw taal (liever niet Assembler ;-)
