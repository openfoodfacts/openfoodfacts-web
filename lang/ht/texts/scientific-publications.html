Piblikasyon Syantifik

Enfòmasyon sou Manje Ouvè nan rechèch: piblikasyon, sitasyon, ak resous pou chèchè yo

Depi 2012, Open Food Facts vin tounen yon resous enpòtan pou chèchè atravè lemond ki itilize done nou yo sou manje pou etid sou nitrisyon, anviwònman, sante piblik ak anpil lòt domèn. Yo site baz done ouvè nou an nan plizyè santèn atik syantifik.

Pou Chèchè yo

Èske w ap itilize oswa w ap konsidere itilize Open Food Facts kòm yon pati nan yon pwojè rechèch?

Tanpri fè nou konnen nan contact@openfoodfacts.org

Nou kontan ede epi kolabore ak ekip rechèch yo pou wè kijan nou ka sipòte travay ou a.

Domèn Rechèch Kle yo

Done nou yo sipòte divès domèn rechèch

Nitrisyon ak Sante Piblik

Chèchè yo itilize baz done nou an pou etidye kalite nitrisyonèl, konpozisyon manje, ak entèvansyon sante piblik. Done Open Food Facts yo te jwe yon wòl enpòtan nan rechèch ak evalyasyon Nutri-Score.

Enpak sou anviwònman an

Etid sou anprent kabòn, Eco-Score, ak dirabilite anviwònmantal pwodui alimantè yo benefisye de baz done konplè nou an ki gen enfòmasyon sou pwodui yo.

Pwosesis Manje

Rechèch sou manje ultra-trete ki itilize klasifikasyon NOVA a baze sou lis engredyan detaye nou yo ak enfòmasyon sou pwosesis yo.

Syans Enfòmatik ak Entèlijans Atifisyèl

Aprantisaj otomatik, rekonesans imaj, tretman lang natirèl, ak pwojè syans done yo itilize ansanm done ak imaj ouvè nou yo.

Konpòtman Konsomatè

Etid ki egzamine chwa manje, efè etikèt, ak transparans nan endistri alimantè a itilize done Open Food Facts.

Politik Manje

Moun ki responsab politik yo ak chèchè yo analize règleman sou manje, egzijans etikèt, ak pratik endistri yo lè l sèvi avèk baz done transparan nou an.

Resous pou Chèchè yo

Kijan pou itilize epi site Open Food Facts

Aksè a Done yo

Open Food Facts bay plizyè fason pou jwenn aksè a done nou yo:

  • Ekspòtasyon baz done: Telechaje tout kopi baz done yo nan plizyè fòma. Gade paj Done nou an pou plis detay.
  • API: Fè rechèch sou baz done nou an atravè pwogramasyon atravè API konplè nou an. Dokimantasyon API a
  • Entèfas rechèch: Eksplore pwodwi yo dirèkteman sou sit entènèt nou an avèk kapasite rechèch avanse ak filtraj.

Lisans

Done nou yo ouvè epi yo ka itilize ankò libman:

  • Baz done a disponib anba Lisans Baz Done Ouvè (ODbL)la
  • Kontni baz done a anba Lisans Kontni Baz Done (DbCL)la
  • Imaj pwodwi yo disponib anba Creative Commons Attribution ShareAlike (CC BY-SA 3.0)

Gade Tèm ak kondisyon itilizasyon ak re-itilizasyon nou yo pou plis detay.

Data Quality

Kòm yon baz done ki soti nan plizyè moun (crowdsourcing), kalite done yo ka varye. Lè w ap itilize Open Food Facts pou rechèch:

  • Konsidere filtraj pa nòt konplè oswa endikatè kalite done yo.
  • Valide pwen done kritik yo si presizyon esansyèl
  • Fè atansyon ak patipri jewografik ak tanporèl nan pwoteksyon done nou yo.
  • Kontakte nou si ou bezwen èd ak evalyasyon kalite done pou rechèch ou an.

Kolabore avèk nou

Nou akeyi kolaborasyon avèk ekip rechèch yo:

  • Bezwen done: Fè nou konnen si ou bezwen done espesifik oswa amelyorasyon nan estrikti baz done nou an
  • Pati eskanè yo: Òganize kanpay koleksyon done siblé pou rejyon rechèch ou an oswa kategori pwodwi ou a
  • Pataj rezilta yo: Pataje rezilta ou yo ak kominote nou an epi ede avanse konesans
  • Devlopman zouti: Kolabore sou zouti ki benefisye ni chèchè yo ni konsomatè yo

Kontakte nou nan contact@openfoodfacts.org pou diskite sou opòtinite kolaborasyon yo.

Enpak Rechèch

Open Food Facts te fè kontribisyon enpòtan nan rechèch syantifik ak sante piblik:

  • Plis pase 600 piblikasyon syantifik te site baz done nou an
  • Gwo ekip rechèch tankou EREN (Frans), USP (Brezil), ak Inivèsite Liège (Bèljik) te itilize done nou yo.
  • Baz done nou an te sipòte rechèch ki mennen nan chanjman nan politik ak rekòmandasyon sante piblik.
  • Nou pèmèt konparezon ant peyi yo ak etid mondyal sou kalite manje ak nitrisyon.
