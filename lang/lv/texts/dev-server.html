Open Food Facts lietotņu un API pamatā esošā informācija

Open Food Facts apkopo datus, izmantojot WRITE API, apstrādā tos un padara pieejamus, izmantojot READ API un tīmekļa versiju. Nutri-Score aprēķināšana, NOVA grupas, piedevu ekstrakcija neatkarīgi no to valodas, sinonīmiem vai iespējamām pareizrakstības kļūdām. Šī ir tikai niecīga daļa no tēmām, ko aptver serveru komanda, lai nodrošinātu pārtikas pārredzamību planētas mērogā.

Piedalieties kodola izstrādē, izmantojot Perl

Vai vēlaties apgūt jaunu valodu? Serveris galvenokārt ir kodēts Perl valodā. Neatkarīgi no tā, vai esat pieredzējis Perl atkarīgais vai tikko esat pieskāries Perl sistēmas administratora uzdevumiem, pastāv kļūda jūsu izmērā un mentori, kas jūs vadīs. Un to var izdarīt vairāk nekā vienā veidā ;-)

Izveidojiet noderīgus rīkus jebkurā valodā

“HungerGames” ir piemērs nelielai spēlei, kas kodēta “React” valodā un pozitīvi ietekmē “Open Food Facts”. Vēl viens rīks LanguageSwitcher ir kodēts Python valodā. Tavs rīks, tava valoda (lūdzu, izvairies no asemblera ;-)

Pievienojieties mums platformās Slack un GitHub

Mēs sinhronizējam pakalpojumā Slack un GitHub. Pievienojies sarunai Open Food Facts Slack vidē (varat pats sevi uzaicināt) un openfoodfacts-server repozitorijā GitHub platformā.
