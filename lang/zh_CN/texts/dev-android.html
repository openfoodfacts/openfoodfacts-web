BugDroid 的甜点问题

我们知道 BugDroid 最喜欢 Nutri-Score 为 E 甜点。 这就是为什么 Android 团队要确保该应用程序对每个人都能良好运行，即使在较旧或更便宜的设备上也是如此。

在 Slack 和 GitHub 上加入 Android 团队

我们在 Slack 和 GitHub 上交流信息。 加入 Open Food Facts slack 上的对话（您可以自行邀请）以及 GitHub 上的 openfoodfacts-androidapp 存储库。

Open Food Facts 的所有志愿者工程团队都在等待您的加入并提出您的第一个拉取请求。 不存在小错误这种情况，您的提交将在几天内（而不是几周）产生巨大影响。
