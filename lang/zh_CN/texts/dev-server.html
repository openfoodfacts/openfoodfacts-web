The intelligence behind the Open Food Facts apps and API

Open Food Facts 使用 WRITE API 收集数据，处理后使用 READ API 提供数据，同时也通过 Web 版本提供数据。 计算 Nutri-Score、NOVA 组、提取添加剂等信息，无论其语言、同义词或潜在的拼写错误。 这只是服务器团队所涵盖主题的一小部分，旨在确保全球范围内的食品透明度。

使用 Perl 为核心做出贡献

想学习一门新语言吗？ 该服务器主要用 Perl 编写。 无论您是经验丰富的 Perl 爱好者，还是刚刚接触 Perl 进行系统管理任务，这里都有适合您的大小的 Bug 供您解决，并且有导师为您提供指导。 而且还有多种方法可以做到这一点;-)

使用任何语言创建有用的工具

HungerGames 是一个用 React 编码的小游戏的例子，它对 Open Food Facts 产生了积极的影响。 另一个工具 LanguageSwitcher 是用 Python 编写的。 您的工具，您的语言（请避免使用汇编程序；-）
