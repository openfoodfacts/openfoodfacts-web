需要专家帮助

研发与发展 Open Food Facts 还需要许多不同领域的专业知识和技能：

  • 项目管理

    我们有很多想法，您可能还有更多想法，但是当每个参与者都是时间有限的志愿者时，确定它们的优先级、制定路线图和管理项目是一个相当大的挑战。

  • 设计

    我们需要帮助在 Open Food Facts 网站和移动应用程序上建立更好的用户体验，改进他们的设计，创建有影响力的演示材料等。

    加入我们的设计工作

  • 开发

    我们有 大量的开发工作要做。 我们使用 Perl 和 MongoDB 作为后端，使用 JSON 格式作为 API 层的数据交换方式，Web 站点部分采用模板化的 HTML5、JavaScript、Foundation 框架，iOS 和 安卓移动应用采用 Flutter 构建。我们同时还构建许多新的工程上可复用的组件。 我们拥有多种编程语言的 SDK 项目，以确保任何人都可以方便地接入和复用以贡献 Open Food Facts。 我们还不断加大人工智能方面的努力，以简化贡献工作的复杂度。 我们的代码托管于 GitHub 。 浏览我们 GitHub 上的项目列表

  • 社区建设

    我们需要在所有国家建立当地社区，同时在全球范围内将它们联合起来。

    如果你不住在一个国家，就很难在那里建立当地社区，所以你的帮助对于找到第一批非常积极的参与者至关重要。 你也是其中之一吗？

  • 通讯

    我们不知道该在这里写些什么，您能帮忙吗？

    更重要的是，用户、贡献者和重复使用者利用 Open Food Facts 做了很多很酷的事情，如果能让更多的人了解它就太好了。 我们需要公众和 媒体关系方面的帮助。

  • 特别项目

    我们有很多有趣且原创的应用，我们希望与政府食品机构、食品生产商、研究人员、大学、学校、非政府组织等合作。 也许你已经有想法了？ 如果您愿意，请帮助我们推动它们前进。
