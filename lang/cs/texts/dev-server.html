Inteligence aplikací a API Open Food Facts

Open Food Facts shromažďuje data pomocí WRITE API, zpracovává je a zpřístupňuje pomocí READ API a webové verze. Počítání Nutri-Score, skupiny NOVA, extrahování přísad, bez ohledu na jejich jazyk, synonyma nebo potenciální pravopisné chyby. Toto je jen malý zlomek témat, kterými se tým serveru zabývá, aby byla zajištěna transparentnost potravin na planetární úrovni.

Přispějte do jádra pomocí Perl

Chcete se naučit nový jazyk? Server je převážně kódován v Perlu. Ať už jste ostřílený fanoušek Perlu, nebo jste se právě dotkli Perlu pro úkoly sys-admin, existuje chyba akorát ve vaší velikosti a mentoři, kteří vás povedou. A také více než jeden způsob, jak to udělat ;-)

Vytvářejte užitečné nástroje v jakémkoli jazyce

HungerGames je příkladem malé hry kódované v reakci, která má pozitivní dopad na Open Food Facts. LanguageSwitcher, další nástroj, je kódován v Pythonu. Váš nástroj, váš jazyk (jen se prosím vyhněte Assembleru ;-)
