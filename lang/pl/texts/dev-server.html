Wiedza kryjąca się za aplikacjami i interfejsem API Open Food Facts

Open Food Facts zbiera dane za pomocą WRITE API, przetwarza je i udostępnia za pomocą API READ i wersji internetowej. Obliczając Nutri-Score, NOVA grupuje, wyodrębniając dodatki, niezależnie od ich języka, synonimów lub potencjalnych błędów ortograficznych. To tylko niewielki ułamek tematów poruszanych przez zespół serwera, aby zapewnić przejrzystość żywności na skalę planetarną.
