Open Food FactsアプリとAPIの背後にあるインテリジェンス

Open Food Facts は、WRITE API を使用してデータを収集し、それを処理して、READ API と Web バージョンを使用して利用できるようにします。 言語、同義語、潜在的なスペルミスに関係なく、Nutri-Score、NOVA グループを計算し、添加物を抽出します。 これは、地球規模で食品の透明性を確保するためにサーバー チームが扱うトピックのほんの一部にすぎません。

Perlを使ってコアに貢献する

新しい言語を学びたいですか? サーバーは主に Perl でコーディングされています。 あなたが熟練した Perl 愛用者であっても、システム管理タスクで Perl に触れたばかりであっても、あなたに合ったサイズのバグがあり、あなたを指導するメンターがいます。 また、それを実行する方法は複数あります ;-)

HungerGames は、React でコーディングされ、Open Food Facts に良い影響を与えている小さなゲームの例です。 別のツールである LanguageSwitcher は Python でコーディングされています。 あなたのツール、あなたの言語 (アセンブラは避けてください ;-)
