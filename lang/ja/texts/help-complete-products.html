製品を完成させ、皆様の食と健康の向上にご協力ください！

あなたが追加する情報は、何百万人もの人々がより良い食品を選択するために、また公衆衛生研究チームの研究のために利用されます。

Nutri-Score を使用して製品の栄養価を判断するには、 NOVA を使用して加工度を分析し、 ビーガンやベジタリアンに適しているかどうかを判断するには、次のことが必要です。

  • 製品を分類する
  • 原材料リストを抽出します (商品写真の原材料リストを選択して切り抜き、 写真からテキストを自動的に抽出することで、非常に簡単に実行できます)
  • 栄養成分表の値を追加する
成分分析：

製品名、ブランド、ラベル（例：オーガニック、フェアトレード）、 原材料の原産地、パッケージの種類などの他の情報を入力することで、さらに多くの機能や比較が可能になります。

あなたが必要です!

Open Food Facts にすでに掲載されている製品の写真やデータを最大限に活用するには、皆様のご協力が必要です。

ご協力いただける方法は次のとおりです:

  1. Open Food Facts Web サイトでアカウント を作成し (モバイル アプリをインストールしている場合は、既にアカウントを作成している可能性があり、そのアカウントは Web サイトでも機能します)、ログインします。
  2. 完了する製品のリストから製品を選択します。
  3. 製品ページの上部にある ページを編集 をクリックします。
  4. 以下の情報を入力してください:
    • 製品名、ブランド、数量などの基本情報を入力してください
    • 製品のカテゴリーを選択してください
    • 製品の写真から原材料リストと栄養成分表を選択して切り取ります。
    • 材料の自動テキスト認識を使用し、コンピューターがすべてを抽出できない場合はスペルを修正します。
    • Nutri-Score を計算できるように、栄養成分表示を入力してください。
  5. Slack に参加して 他の貢献者とチャットしたり交流したり、質問したり、私たちの特別な作戦に参加したりしましょう!

材料の抽出には数秒しかかかりません。

ありがとう！

Open Food Facts は、皆様のご協力がなければ存在し得ない非営利プロジェクトです。 私たちが食べる食べ物に関するデータはさまざまな用途があり、多くの人の生活を 変えることができます。 This is why all the data we gather is freely available to all, in open data, so that we can build a common good that benefits everyone.

その結果、あなたの貢献は、当社のモバイル アプリや Web サイトを使用する何百万人もの人々にとって、より良い食品を選択するのに役立ちます また、当社のデータを研究に使用する公衆衛生研究者にも役立ちます。
