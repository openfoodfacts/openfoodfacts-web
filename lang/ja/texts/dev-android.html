BugDroidのデザート問題

BugDroid は主に NutriScore E デザートを好むことがわかっています。 そのため、Android チームは、古いデバイスや安価なデバイスでも、すべてのユーザーにとってアプリが問題なく動作することを保証しています。

Slack と GitHub で Android チームに参加しましょう

Slack と GitHub で同期します。 Open Food Facts の Slack (自分で招待できます) と GitHub の openfoodfacts-androidapp リポジトリでの会話 に参加してください。

Open Food Facts のボランティア エンジニアリング チーム全員が、あなたの参加と最初のプル リクエストの作成を待っています。 小さなバグというものは存在せず、あなたのコミットは数週間ではなく数日で大きな影響を与えることになります。
