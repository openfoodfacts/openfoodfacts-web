Интеллектуальные возможности приложений и API Open Food Facts

Open Food Facts собирает данные с помощью WRITE API, обрабатывает их и делает доступными с помощью READ API и веб-версии. Вычисление индекса Nutri-Score, групп NOVA, извлечение добавок независимо от их языка, синонимов или возможных ошибок в написании. Это лишь малая часть тем, которые освещает серверная команда, чтобы обеспечить прозрачность в вопросах продовольствия в планетарном масштабе.

Внесите свой вклад в ядро, используя Perl

Хотите выучить новый язык? Сервер написан в основном на Perl. Независимо от того, являетесь ли вы опытным пользователем Perl или только начинаете использовать его для решения задач системного администратора, для вас найдется ошибка, подходящая именно вам, и наставники, которые вам помогут. И сделать это можно множеством разных способов ;-)

Создавайте полезные инструменты на любом языке

«Голодные игры» — это пример небольшой игры, написанной на React, которая оказывает положительное влияние на Open Food Facts. LanguageSwitcher — другой инструмент написанный на Python. Ваш инструмент, ваш язык (пожалуйста, избегайте Assembler ;-)
