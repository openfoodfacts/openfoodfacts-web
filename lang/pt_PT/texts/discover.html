Uma base de dados de produtos alimentares

O Open Food Facts é uma base de dados de produtos alimentares com ingredientes, alérgenos, informações nutricionais e todos os fragmentos de informações que podemos encontrar nos rótulos dos produtos.

Feito por todos

A Open Food Facts é uma associação sem fins lucrativos de voluntários.

Mais de 15.000 colaboradores adicionaram mais de 1 000 000 produtos de 150 países usando a nossa aplicação ,iPhone ou Windows Phone ou a sua câmara para ler códigos de barras e enviar fotografias de produtos e respetivos rótulos.

Para todos

Os dados sobre comida são de interesse público e devem ser abertos. A base de dados completa é publicada como dados abertos e pode ser reutilizada por qualquer pessoa e para qualquer utilização. Veja as reutilizações interessantes ou faça suas próprias!

Faça melhores escolhas alimentares

Conhecimento é poder. Conhecimento alimentar é poder para comer melhor!

  • Descodifique rótulos

    Aditivos alimentares com números E, alérgenos, códigos de embalagem... O Open Food Facts ajuda a entender as letras miúdas dos rótulos dos produtos.

    Também classificamos a qualidade nutricional de cada produto.

  • Encontre produtos

    O nosso formulário de pesquisa inclui dezenas de critérios para que possa encontrar produtos que correspondam aos seus critérios.

  • Compare e mude

    Sabe quais os cereais de pequeno almoço têm menos gordura e açúcar? Em 3 cliques, pode criar um gráfico interativo para mostrá-lo. Desta forma pode mudar para cereais mais saudáveis e ao comparar produtos de marcas diferentes incentiva os produtores a torná-los melhores.

  • Explore e descubra

    Curioso sobre comida? Explore os produtos por facetas, como marcas, categorias, rótulos, origens dos ingredientes etc. As facetas podem ser combinadas para descobrir novas perspetivas, por exemplo, para mapear as origens de ingredientes que têm o rótulo de comércio justo.

Alimento para o pensamento, inovação e ciência

Os nossos dados podem ser reutilizados para educação, novos produtos e serviços, pesquisas e muito mais!

A base de dados do Open Food Facts são dados abertos publicados sob a Open Database License. Qualquer pessoa pode utilizá-lo para qualquer finalidade - comercial ou não comercial - desde que a fonte dos dados seja atribuída e as melhorias na base de dados sejam partilhadas de forma semelhante.
