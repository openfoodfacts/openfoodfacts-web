Publicações científicas

Informação aberta sobre alimentos na investigação: publicações, citações e recursos para investigadores.

Desde 2012, o Open Food Facts tornou-se um recurso valioso para investigadores de todo o mundo que utilizam os nossos dados alimentares em estudos sobre nutrição, ambiente, saúde pública e muitas outras áreas. A nossa base de dados aberta foi citada em centenas de artigos científicos.

Principais áreas de investigação

Os nossos dados dão suporte a diversas áreas de investigação.

Nutrição e Saúde Pública

Os investigadores utilizam a nossa base de dados para estudar a qualidade nutricional, a composição dos alimentos e as intervenções em saúde pública. Os dados do Open Food Facts têm sido fundamentais na investigação e avaliação do Nutri-Score.

Impacto ambiental

Os estudos sobre a pegada de carbono, Eco-Score e sustentabilidade ambiental dos produtos alimentares beneficiam da nossa base de dados abrangente de informações sobre produtos.

Processamento de Alimentos

A investigação sobre alimentos ultraprocessados, utilizando a classificação NOVA, baseia-se nas nossas listas detalhadas de ingredientes e informações sobre o processamento.

Ciência da Computação e IA

Os projetos de aprendizagem automática, reconhecimento de imagem, processamento de linguagem natural e ciência de dados utilizam os nossos conjuntos de dados e imagens abertos.

Comportamento do consumidor

Os estudos que examinam as escolhas alimentares, os efeitos da rotulagem e a transparência na indústria alimentar utilizam dados da Open Food Facts.

Política Alimentar

Os decisores políticos e os investigadores analisam as regulamentações alimentares, os requisitos de rotulagem e as práticas da indústria utilizando a nossa base de dados transparente.

Recursos para investigadores

Como usar e citar o Open Food Facts

Acesso aos dados

O Open Food Facts oferece diversas formas de aceder aos nossos dados:

  • Exportações de base de dados: Descarregue dumps completos da base de dados em vários formatos. Consulte nossa página Dados para mais detalhes.
  • API: Consulte a nossa base de dados programaticamente através da nossa API abrangente. Documentação da API
  • Interface de pesquisa: Explore os produtos diretamente no nosso website com funcionalidades avançadas de pesquisa e filtragem.

Citando o Open Food Facts

Se utilizar o Open Food Facts na sua pesquisa, por favor, cite-o adequadamente. Segue uma sugestão de formato de citação:

Citação do site:

Informações alimentares claras. (Ano). Informação Aberta sobre Alimentos - Mundo. Retirado de https://world.openfoodfacts.org


Citação do conjunto de dados:

Informações alimentares claras. (Ano). Base de dados Open Food Facts [Conjunto de dados]. Retirado de https://world.openfoodfacts.org/data

Licenciamento

Os nossos dados são abertos e podem ser reutilizados livremente:

  • A base de dados está disponível sob a Open Database License (ODbL)
  • O conteúdo da base de dados está sob a Licença de Conteúdo da Base de Dados (DbCL)

Consulte os nossos Termos e condições de utilização e reutilização para obter todos os detalhes.

Qualidade dos dados

Por se tratar de uma base de dados colaborativa, a qualidade dos dados pode variar. Ao utilizar o Open Food Facts para pesquisa:

  • Considere filtrar por pontuações de completude ou indicadores de qualidade de dados.
  • Valide os pontos de dados críticos se a precisão for essencial.
  • Esteja ciente dos enviesamentos geográficos e temporais na nossa cobertura de dados.
  • Contacte-nos se precisar de ajuda com a avaliação da qualidade dos dados para a sua investigação.

Colabore connosco

Agradecemos a colaboração com as equipas de investigação:

  • Necessidades de dados: Informe-nos se necessita de dados específicos ou melhorias na estrutura da nossa base de dados
  • Equipas de scan: Organize campanhas de recolha de dados direcionadas para a sua região de pesquisa ou categoria de produto
  • Partilha de resultados: Partilhe as suas descobertas com a nossa comunidade e ajude a fazer avançar o conhecimento
  • Desenvolvimento de ferramentas: Colaborar no desenvolvimento de ferramentas que beneficiem tanto os investigadores como os consumidores.

Contacte-nos através do endereço contact@openfoodfacts.org para discutir oportunidades de colaboração.

Impacto da investigação

O projeto Open Food Facts tem dado contributos significativos para a investigação científica e para a saúde pública:

  • Mais de 600 publicações científicas citaram a nossa base de dados.
  • Grandes equipas de investigação, incluindo a EREN (França), a USP (Brasil) e a Universidade de Liège (Bélgica), utilizaram os nossos dados.
  • A nossa base de dados tem apoiado pesquisas que levaram a mudanças nas políticas públicas e a recomendações de saúde pública.
  • Possibilitamos comparações entre países e estudos globais sobre qualidade alimentar e nutrição.
