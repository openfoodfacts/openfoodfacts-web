Expert help needed

Open Pet Food Facts хатӗрлесси те нумай тӗрлӗ енӗпе ятарлӑ пӗлӳпе ӑсталӑх ыйтать:

  • Project management

    We have tons of ideas and you probably have even more, but it is quite a challenge to prioritize them, to build a roadmap, and to manage projects when every participant is a volunteer with often limited time available.

  • Design

    We need help to build a better user experience on the Open Pet Food Facts web site and mobile app, to improve their design, to create impactful presentation materials etc.

  • Development

    Пирӗн аталану ӗҫӗсем нумай: Open Pet Food Facts бекэндӗнче (Perl тата MongoDB), API (JSON), веб-сайт ҫинче (HTML5, JS, Foundation) тата мобильлӗ приложени ҫинче. Our code is on GitHub.

  • Community building

    We need to build local communities in all countries and at the same time unite them globally.

    It is very difficult to bootstrap a local community in a country without living there, so your help to find the first very motivated participants is essential. Are you one of them?

  • Communication

    We are not sure what to write here, could you help?

    Чӑнах та, Open Pet Food Facts-па усӑ куракансем, хутшӑнакансем тата тепӗр хут усӑ куракансем нумай лайӑх япаласем тӑваҫҫӗ, ун ҫинчен ытларах ҫынна пӗлтерсен питӗ аван пулӗччӗ. In particular, we need help for public and media relations.

  • Special projects

    Килти чӗрчунсен апат-ҫимӗҫӗн уҫӑ даннӑйӗсен нумай интереслӗ те хӑйне евӗрлӗ приложенийӗсем пур, вӗсем ҫинче эпир ветеринарсемпе, чӗрчунсен диетологӗсемпе, чӗрчунсен апат-ҫимӗҫне туса кӑларакансемпе, тӗпчевҫӗсемпе, университетсемпе, шкулсемпе, чӗрчунсене хӳтӗлекен НКОсемпе тата ыттисемпе ӗҫлеме пултаратпӑр. Maybe you already have ideas? If you do, please help us to push them forward.
