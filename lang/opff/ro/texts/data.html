API JSON live

API-ul se comportă în proporție de 99% ca Open Food Facts, așa că multe dintre SDK-urile, codurile pentru a interacționa cu API-ul Open Food Facts ar trebui să funcționeze prin simpla schimbare a punctului final de la: openfoodfacts.org la openpetfoodfacts.org. Codul nu a fost actualizat de peste un an, așa că multe dintre API-urile mai noi, cum ar fi Knowledge Panels, nu sunt încă disponibile.

Un API JSON este, de asemenea, disponibil pentru a citi datele unui produs. Acest API este utilizat în special în aplicația mobilă Open Pet Food Facts pentru iPhone și Android.

Schimbăm API-ul în timp, dar avem grijă să păstrăm compatibilitatea cu versiunea anterioară. Vă rugăm să ne spuneți dacă îl utilizați, astfel încât să vă putem spune despre eventualele modificări.

URL pentru a citi datele pentru un produs: https://world.openpetfoodfacts.org/api/v0/product/[barcode].json

Exemplu: https://world.openpetfoodfacts.org/api/v0/product/3600550578041.json

  • API V1- Documentație veche disponibilă pe Wiki (Notă importantă: pagina este împărțită în subpagini pentru CITIRE și SCRIERE).
  • API V2 - Cuprinzător, dar nu compatibil OpenAPI - Conține cazuri de utilizare pentru afișarea unui produs și permite utilizatorilor să adauge datele necesare pentru a obține Nutri-Score, clasificarea NOVA, aditivi și Eco-Score.
  • API V2 - noua documentație OpenAPI - Lucru în desfășurare
  • API V3 - noua documentație OpenAPI - Lucru în desfășurare

Sunteți binevenit să utilizați API-ul pentru cazurile de producție, atâta timp cât 1 apel API = 1 scanare reală de către un utilizator. Orice încercare de a răzui baza de date folosind API-ul va fi foarte probabil blocată, deoarece exporturile zilnice complete sunt disponibile chiar pe această pagină.

De asemenea, trimiterea unui antet HTTP cu apelul dvs. API este o modalitate bună prin care putem contacta, în cazul în care apare orice problemă.

Dacă nu sunteți sigur de ceva sau aveți întrebări (nu există nicio întrebare prostească), nu ezitați să o întrebați pe Slack :-)

Aplicație mobilă pentru Android și iPhone (este nevoie de ajutor!)

Codul pentru aplicația mobilă Open Pet Food Facts implementată în prezent este disponibil pe GitHub (Android. Avem nevoie ajutor pentru a adapta noul cod Flutter la hrana pentru animale de companie. Aplicația permite utilizatorilor să scaneze codul de bare al produselor, să vadă informațiile despre produs și să facă și să trimită fotografii și date pentru produsele lipsă.

Aplicația pentru Android este dezvoltată în Kotlin.

Aplicația clasică pentru iOS nu a fost niciodată adaptată și nici implementată pentru hrana pentru animale de companie.

Noua aplicație multiplatformă este dezvoltată în Flutter/Dart.

Sunteți bineveniți să reutilizați codul (este open source) și să ne ajutați să îl îmbunătățim pentru toată lumea, pe întreaga planetă.
