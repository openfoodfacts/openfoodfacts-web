Una base de datos de productos alimenticios para mascotas

Open Pet Food Facts es una base de datos de productos de alimentación para mascotas con ingredientes, valores nutricionales y toda la información que podemos encontrar en las etiquetas de los productos.

Hecho por todos

Open Pet Food Facts es una asociación sin ánimo de lucro formada por voluntarios.

Los dueños de mascotas como usted pueden agregar productos de todo el mundo usando la aplicación Open Food Facts para Android o iPhone o su cámara para escanear códigos de barras y subir fotos de los productos y sus etiquetas.

Descarga la app para empezar a escanear comida para mascotas.

Para todos

Los datos sobre alimentos para mascotas son de interés público y deben ser transparentes. Toda la base de datos se publica como datos abiertos y puede ser reutilizada por cualquiera y para cualquier uso. ¡Echa un vistazo a las reutilizaciones geniales o crea la tuya propia!

Elige mejor la comida para tu mascota

El conocimiento es poder. ¡El conocimiento sobre alimentación para mascotas es poder para alimentar mejor a tus mascotas!

  • Descodificar etiquetas

    Ingredientes, aditivos y valores nutricionales de los alimentos para mascotas... Open Pet Food Facts te ayuda a entender la letra pequeña de las etiquetas de los productos.

  • Encontrar productos

    Nuestro formulario de búsqueda incluye docenas de criterios para que pueda encontrar productos de comida para mascotas que se ajusten a las necesidades de su mascota.

  • Compara y elige

    ¿Sabes qué alimentos para perros tienen más proteínas? ¿Qué alimentos para gatos no contienen cereales? Puedes comparar productos de diferentes marcas y tomar decisiones informadas para la salud de tus mascotas.

  • Explorar y descubrir

    ¿Sientes curiosidad por la comida para mascotas? Explora los productos por aspectos como marcas, categorías, etiquetas, especies (perros, gatos, etc.) e ingredientes. Descubre qué contiene realmente la comida de tu mascota.

Alimentos para mascotas para la innovación y la ciencia

¡Nuestros datos se pueden reutilizar para el ámbito educativo, la elaboración de nuevos productos y servicios, la investigación y más!

La base de datos Open Pet Food Facts es información abierta publicada bajo la Licencia de Base de Datos Abierta. Cualquier persona lo puede usar para cualquier propósito -comercial o no comercial- si se atribuyen la fuente de los datos y las mejoras a la base de datos son compartidas de forma similar.

Educación

Aprende sobre nutrición para mascotas y comprende qué constituye una dieta saludable para tu perro, gato u otras mascotas. Los datos abiertos ayudan a los dueños de mascotas a tomar decisiones informadas.

Productos y servicios

Los datos de Open Pet Food Facts pueden reutilizarse tanto en proyectos no comerciales como comerciales, siempre y cuando también se compartan las mejoras realizadas a la base de datos.

Ciencia

Los datos de Open Pet Food Facts pueden ser utilizados por veterinarios, nutricionistas de mascotas e investigadores que estudian la nutrición y la salud animal. ¡Contribuye a mejorar la ciencia de los alimentos para mascotas!
