Умови повторного використання

База даних Open Pet Food Facts доступна за ліцензією Open Database License. Окремі частини бази даних доступні за ліцензією на вміст бази даних. Зображення продуктів доступні за ліцензією Creative Commons Attribution ShareAlike. Вони можуть містити графічні елементи, що підлягають авторському праву чи іншим правам, які можуть у деяких випадках бути відтворені (права на цитування або добросовісне використання).

Будь ласка, перед повторним використанням даних ознайомтеся з Умовами та положеннями використання та повторного використання.

Дамп та експорт бази даних генерується щоночі. База даних дуже велика, тому ви можете знайти різні способи використання даних, залежно від особливостей експорту.

Інформація про різні поля для дамбу MongoDB та експорту CSV доступна за адресою https://world.openpetfoodfacts.org/data/data-fields.txt.

Експорт дельти

Щоденний експорт дельти представлені на попередні 14 днів. Список доступних наразі дельта-файлів можна знайти на https://static.openpetfoodfacts.org/data/delta/index.txt.

Кожен рядок представляє файл, доступний за адресою https://static.openpetfoodfacts.org/data/delta/{filename}.

Ім'я файлу містить UNIX-мітку часу першої та останньої зміни, що міститься у JSON-файлі, так що дельта-файли можна імпортувати (після вилучення) за допомогою mongoimport в алфавітному порядку.

Будь ласка зауважте, що через природу mongoexport, дельти файлів не можуть розповісти вам про видалені продукти. Щоб вилучити видалені продути з вашої бази даних, вам буде потрібно імпортувати повний дамп MongoDB.

Експорт даних JSONL

Повна база даних є також наявною в форматі JSONL (ще назвають LDJSON чи NDJSON) де кожний рядок є об'єктом JSON. Вона представляє такі ж самі дані як експорт MongoDB. Файл стискається за допомогою gzip.

Силка
https://static.openpetfoodfacts.org/data/openpetfoodfacts-products.jsonl.gz

Відповідним способом використання бази даних є DuckDB, аналітичний інструмент, що працює всередині процесу і призначений для обробки великих обсягів даних за частку секунди. Ви можете прочитати наш допис у блозі, де ми ознайомимо вас із вивченням та обробкою бази даних Open Pet Food Facts за допомогою DuckDB

Експорт даних RDF

Дані для всіх або деяких продуктів можна завантажити у форматі CSV (можна прочитати за допомогою LibreOffice, Excel та багатьох інших програм для роботи з електронними таблицями) через форму розширеного пошуку.

Силка
https://static.openpetfoodfacts.org/data/en.openpetfoodfacts.org.products.csv.gz (стиснутий CSV у форматі GZIP: ~ 0,9 Гб, нестиснутий: ~ 9 Гб)

Кодування файлу є юнікодом UTF-8. Символ, що розділяє поля є <tab> (табуляція).

Експериментальний XML API

Ми прагнемо закінчити документування API, додати кейси використання та автоматизувати генерацію SDK. Якщо ви знавець OpenAPI, ми будемо раді вашій допомозі.

Ми дуже охочі до використання API для виробничих кейсів, доки 1 API = 1 справжньому скануванню користувача. Будь-яка спроба здерти базу даних використовуючи API буде ймовірно заблокованою, так як повний щоденний експорт саме на цій сторінці.

Також, надсилання HTTP заголовку з вашим викликом API є хорошим способом для нашого зв'язку, якщо виникне будь-яке питання.

Вони дозволять вам споживати дані, так само і дозволять іншим користувачам вносити нові дані

Обкладинки та SDK

Ми можливо маємо обкладинку для вашої улюбленої мови програмування. Якщо так, то ви можете використовувати її та покращувати. Якщо ні, ви можете допомогти створити її.

Вітається повторне використання вами коду (це відкрите джерело) та допомога нам у його покращенні, для кожного, по всій планеті.

Якщо ваші користувачі не очікують результат одразу (на приклад додатки для інвентиризації)

  • Ви можете шукати інформацію про товари, включаючи багато корисних розрахункових значень.
  • Якщо ви не можете отримати інформацію про певний продукт, ви можете попросити свого користувача надіслати фото та дані, які потім оброблять Open Pet Food Facts AI та співавтори, щоб отримати обчислений результат, який ви хочете їм показати.
  • Ви також можете реалізувати цілий потік, щоб вони одразу отримали результат з деякими зусиллями з їхнього боку.

Якщо ваші користувачі очікують результат одразу (на приклад програми щодо харчування)

  • Надсилайте фото (передня частина/харчування/інгредієнти): найбезболісніша річ для ваших користувачів
  • ШІ Open Pet Food Facts – Robotoff – згенерує деякі похідні дані з фотографій
  • Згодом інші програми та спільнота Open Beauty Facts заповнять прогалини в даних

Якщо ваші користувачі очікують на негайний результат (наприклад, програми для харчування чи алергії)

  • Надіслати складники + категорію > отримати аналіз складників, добавки, алергени, нормалізовані складники, веганство, вегетаріанство…
  • Надішліть етикетки для додаткової вартості
  • Подати харчування
