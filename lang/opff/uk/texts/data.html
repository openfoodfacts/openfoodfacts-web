Зображення продуктів доступні за ліцензією Creative Commons Attribution ShareAlike. Вони можуть містити графічні елементи, що підлягають авторському праву чи іншим правам, які можуть у деяких випадках бути відтворені (права на цитування або добросовісне використання).

Будь ласка, перед повторним використанням даних ознайомтеся з Умовами та положеннями використання та повторного використання.

Нам дуже цікаво дізнатися, для чого використовуються дані Open Pet Food Facts. Це не є обов'язковим, але ми будемо дуже вдячні, якщо ви розповісте нам про своє повторне використання, щоб ми могли поділитися цим зі спільнотою Open Pet Food Facts.

Експорт дельти

Щоденний експорт дельти представлені на попередні 14 днів. Список доступних наразі дельта-файлів можна знайти за посиланням https://static.openpetfoodfacts.org/data/delta/index.txt. Кожен рядок представляє файл, доступний за адресою https://static.openpetfoodfacts.org/data/delta/{filename}. Назва файлу містить часову позначку UNIX на першій та останній зміні, що міститься у файлі JSON, таким чином дельта файли можуть бути імпортовані (після витягнення) з mongoimport у алфебетному порядку.

Будь ласка зауважте, що через природу mongoexport, дельти файлів не можуть розповісти вам про видалені продукти. Щоб вилучити видалені продукти з вашої бази даних, вам буде потрібно імпортувати цілий MongoDB dump.

Діючий API JSON

The API behaves 99% like Open Food Facts, so many of the SDKs, codes to interact with the Open Food Facts API should work by merely changing the endpoint from: openfoodfacts.org to openpetfoodfacts.org. The code hasn't been updated in over a year, so many of the newer APIs like Knowledge Panels are not available yet.

API JSON також є наявним для читання даних продукту. Цей API, зокрема, використовується в мобільному додатку Open Pet Food Facts для iPhone та Android.

Ми з часом змінюємо API, проте ми ставимося з обережністю до підтримання зворотньої сумісності. Будь ласка розкажіть нам чи ви використовуєте це, таким чином ми зможемо розповісти вам про наші потенційні зміни.

URL для читання даних про продукт: https://world.openpetfoodfacts.org/api/v0/product/[barcode].json

Приклад: https://world.openpetfoodfacts.org/api/v0/product/3600550578041.json

  • API V1 - Стара документація доступна на Вікі (Важливе зауваження: сторінка розділена на підсторінки для ЧИТАННЯ і ЗАПИСУ).
  • API V2 - повний, але не сумісний з OpenAPI - містить варіанти використання для відображення продукту і дозволяє вашим користувачам додавати дані, необхідні для отримання Nutri-Score, класифікації NOVA, добавок та Eco-Score.
  • API V2 - нова документація OpenAPI - Work in progress
  • API V3 - нова документація по OpenAPI - Робота в процесі

Ми дуже охочі до використання API для виробничих кейсів, доки 1 API = 1 справжньому скануванню користувача. Будь-яка спроба здерти базу даних використовуючи API буде ймовірно заблокованою, так як повний щоденний експорт саме на цій сторінці.

Також, надсилання HTTP заголовку з вашим викликом API є хорошим способом для нашого зв'язку, якщо виникне будь-яке питання.

Якщо ви непевні щодо чогось, або маєте якісь запитання (немає дурних запитань), вільно почувайтеся спитати їх у Slack :-)

Android and iPhone mobile app (help needed!)

The code for the Open Pet Food Facts mobile app currently deployed is available on GitHub (Android. Нам потрібна допомога, щоб адаптувати новий код Flutter до кормів для тварин. Додаток дозволяє користувачам сканувати штрих-код продуктів, переглядати інформацію про продукт, а також фотографувати та надсилати фотографії та дані про відсутні продукти.

The app for Android is developed in Kotlin.

The classic app for iOS has never been adapted nor deployed for pet food.

The new cross-platform app is developed in Flutter/Dart.

Вітається повторне використання вами коду (це відкрите джерело) та допомога нам у його покращенні, для кожного, по всій планеті.
