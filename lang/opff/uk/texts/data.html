Діючий API JSON

API поводиться на 99% як Open Food Facts, тому багато SDK, кодів для взаємодії з Open Food Facts API повинні працювати, просто змінивши кінцеву точку з: openfoodfacts.org на openpetfoodfacts.org. Код не оновлювався понад рік, тому багато новіших API, як-от панелі знань, ще недоступні.

API JSON також є наявним для читання даних продукту. Цей API, зокрема, використовується в мобільному додатку Open Pet Food Facts для iPhone та Android.

Ми з часом змінюємо API, проте ми ставимося з обережністю до підтримання зворотньої сумісності. Будь ласка розкажіть нам чи ви використовуєте це, таким чином ми зможемо розповісти вам про наші потенційні зміни.

URL для читання даних про продукт: https://world.openpetfoodfacts.org/api/v0/product/[barcode].json

Приклад: https://world.openpetfoodfacts.org/api/v0/product/3600550578041.json

  • API V1 - Стара документація доступна на Вікі (Важливе зауваження: сторінка розділена на підсторінки для ЧИТАННЯ і ЗАПИСУ).
  • API V2 - повний, але не сумісний з OpenAPI - містить варіанти використання для відображення продукту і дозволяє вашим користувачам додавати дані, необхідні для отримання Nutri-Score, класифікації NOVA, добавок та Eco-Score.
  • API V2 - нова документація OpenAPI - Work in progress
  • API V3 - нова документація по OpenAPI - Робота в процесі

Ми дуже охочі до використання API для виробничих кейсів, доки 1 API = 1 справжньому скануванню користувача. Будь-яка спроба здерти базу даних використовуючи API буде ймовірно заблокованою, так як повний щоденний експорт саме на цій сторінці.

Також, надсилання HTTP заголовку з вашим викликом API є хорошим способом для нашого зв'язку, якщо виникне будь-яке питання.

Якщо ви непевні щодо чогось, або маєте якісь запитання (немає дурних запитань), вільно почувайтеся спитати їх у Slack :-)

Мобільний додаток для Android та iPhone (потрібна допомога!)

Код мобільного додатку Open Pet Food Facts, який наразі розгорнуто, доступний на GitHub (Android). Нам потрібна допомога, щоб адаптувати новий код Flutter до кормів для тварин. Додаток дозволяє користувачам сканувати штрих-код продуктів, переглядати інформацію про продукт, а також фотографувати та надсилати фотографії та дані про відсутні продукти.

Додаток для Android розроблений у Kotlin.

Класичний додаток для iOS ніколи не був адаптований або розгорнутий для кормів для домашніх тварин.

Вітається повторне використання вами коду (це відкрите джерело) та допомога нам у його покращенні, для кожного, по всій планеті.
