Uudelleenkäytön ehdot

Open Pet Food Facts -tietokanta on saatavilla Open Database -lisenssillä.
Tietokannan yksittäinen sisältö on saatavilla Tietokannan sisältölisenssillä.
Tuotteen kuvat ovat saatavilla Creative Commons Attribution ShareAlike -lisenssillä. Ne saattavat sisältää tekijänoikeuden tai muun oikeuden alaisia graafisia elementtejä, jotka joissain tapauksissa saa kopioida (lainausoikeudet tai kohtuullinen käyttö).

Lue Käyttöehdot ja uudelleenkäyttö ennen tietojen uudelleenkäyttöä.

Kerro meille uudelleenkäytöstäsi

Olemme erittäin kiinnostuneita oppimaan, mihin Open Pet Food Facts -tietoja käytetään. It is not mandatory, but we would very much appreciate it if you tell us about your re-uses so that we can share them with the Open Pet Food Facts community. Voit myös täyttää tämän lomakkeen saadaksesi mahdollisuuden saada sovelluksesi esille.

Android- ja iPhone-mobiilisovellus (apua tarvitaan!)

Tietokantavedokset ja -viennit luodaan öisin. Tietokanta on erittäin suuri, joten voit löytää erilaisia tapoja käyttää tietojavientimausta riippuen.

Tietoja MongoDB-vedosten ja CSV-vientien eri kentistä on saatavilla osoitteessa https://world-opff.openpetfoodfacts.org/data/data-fields.txt

Muutosten vienti

Päivittäiset muutosten viennit ovat saatavilla viimeiseltä 14:ltä päivältä. The list of currently available delta files can be found at https://static.openpetfoodfacts.org/data/delta/index.txt.

Each line represents a file that is available at https://static.openpetfoodfacts.org/data/delta/{filename}.

Tiedostonimi sisältää JSON-tiedoston ensimmäisen ja viimeisen muutoksen UNIX-aikaleiman, jotta delta-tiedostot voidaan tuoda (purkauksen jälkeen) mongoimport :lla aakkosjärjestyksessä.

Huomioi, että mongoexport-komennon luonteen takia muutostiedostot eivät voi kertoa poistetuista tuotteista. Jos haluat poistaa poistetut tuotteet tietokannastasi, sinun on tuotava koko MongoDB-vedos.

JSONL-tietojen vienti

Koko tietokanta on saatavilla myös JSONL-muodossa (joskus kutsutaan nimellä LDJSON tai NDJSON), jossa jokainen rivi on JSON-objekti. Se edustaa samoja tietoja kuin MongoDB-vienti. Tiedosto on pakattu gzip:llä.

Linkki
https://static.openpetfoodfacts.org/data/openpetfoodfacts-products.jsonl.gz

A suitable way to exploit the database is to use DuckDB, an in-processs analytical tool designed to process large amount of data in a fraction of seconds. You can read our blog post where we walk you through exploring and processing the Open Pet Food Facts database with DuckDB

Tietojen CSV-vienti

Data for all products, or some of the products, can be downloaded in the CSV format (readable with LibreOffice, Excel and many other spreadsheet software) through the advanced search form.

Linkkejä
https://static.openpetfoodfacts.org/data/en.openpetfoodfacts.org.products.csv.gz (compressed CSV in GZIP format: ~ 0.9 Gb, uncompressed: ~ 9 Gb)

Tiedoston merkistökoodaus on Unicode UTF-8. Kenttien erottimena on <sarkain>.

Kuvatietojen vienti

Tietokanta on myös saatavilla RDF-muodossa. Voit lukea ilmoituksen ranskaksi.

OpenAPI-apua kaivataan

Kaikki kuvat ja OCR-tulokset voidaan ladata joko palvelimeltamme tai AWS:stä (AWS Open Data Program -ohjelman kautta). Tutustu dokumentaatioomme saadaksesi lisätietoja miten ladata kuvia.

Live JSON ja XML API

JSON API on myös saatavilla tuotteen tietojen lukemiseen. Tätä sovellusliittymää käytetään erityisesti Open Pet Food Facts -mobiilisovelluksessa iPhonelle ja Androidille.

Parannamme ohjelmointirajapintaa ajan myötä, mutta huolehdimme yhteensopivuudesta aiempien versioiden kanssa. Please get in touch if you use it, so that we can tell you about potential changes.

Jos olet epävarma jostakin, tai sinulla on kysymyksiä (hölmöjä kysymyksiä ei ole), voit esittää kysymyksesi Slackissä :-)

Example: https://world-opff.openpetfoodfacts.org/api/v3/product/737628064502.json

Kokeellinen XML API

Pyrimme saamaan valmiiksi API:n dokumentaation, lisäämään käyttökokemusta ja automatisoimaan SDK-generaatiota. Jos olet OpenAPI-asiantuntija, olet hyvin tervetullut auttamaan.

Voit käyttää API:a tuotantotapauksia varten, kunhan yksi API-kutsu edellyttää yhtä aitoa skannausta käyttäjän toimesta. Kaikki yritykset tietokannan kopioimiseksi API:n avulla todennäköisesti estetään, koska kaikki päivittäiset tietoviennit ovat saatavilla tällä sivulla.

Tämän lisäksi HTTP-päätteen lähettäminen API-soiton yhteydessä on hyvä tapa ottaa meihin yhteyttä, jos ilmenee ongelmia.

Voit liittyä Open Food Facts Slack -keskusteluryhmään, joka on ensisijainen tapa kysyä kysymyksiä ja keskustella API:sta.

Kääreet ja SDK:t

Meillä on todennäköisesti kääre lempiohjelmointikielellesi. Jos on, voit käyttää ja parantaa sitä. Jos ei ole, voit auttaa luomaan sen.

Voit uudelleen käyttää koodia (se on avointa lähdekoodia), ja auttaa meitä parantamaan sitä, kaikille, koko planeetalla.

Yleiset periaatteet käyttäjien ilahduttamiseksi

  • Voit etsiä tietoja tuotteista, mukaan lukien monia hyödyllisiä laskettuja arvoja.
  • DART: GitHub (vain Open Food Facts tällä hetkellä)
  • Voit myös toteuttaa koko flown niin, että he saavat heti tuloksen pienellä vaivalla.

Jos käyttäjäsi eivät odota tulosta heti (esim. varastosovellukset)

  • Lähetä valokuvat (etu/ravinto/ainesosat): käyttäjillesi kivuton asia
  • The Open Pet Food Facts AI - Robotoff - will generate some derived data from the photos
  • Over time, other apps, and the Open Pet Food Facts community will fill the data gaps

If your users expect a result immediately (e.g. Nutrition or Allergy apps)

  • Submit ingredients + category > get Ingredient analysis, additives, allergens, normalized ingredients, vegan, vegetarian…
  • Submit labels for added value
  • Submit nutrition
