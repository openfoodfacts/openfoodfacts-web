API JSON langsung

API ini berperilaku 99% seperti Open Food Facts, sehingga banyak SDK, kode untuk berinteraksi dengan API Open Food Facts dapat digunakan hanya dengan mengubah titik akhir dari: openfoodfacts.org menjadi openpetfoodfacts.org. Kode ini belum diperbarui selama lebih dari satu tahun, sehingga banyak dari API yang lebih baru seperti Panel Pengetahuan belum tersedia.

API JSON juga tersedia untuk membaca data suatu produk. API ini secara khusus digunakan dalam aplikasi seluler Open Pet Food Facts untuk iPhone dan Android.

Kami mengubah API dari waktu ke waktu, tapi kami berhati - hati untuk menjaga kompatibilitas ke belakangnya. Harap memberitahu kami jika Anda menggunakannya, sehingga kami dapat memberi tahu Anda tentang kemungkinan perubahannya.

URL untuk membaca data suatu produk: https://world.openpetfoodfacts.org/api/v0/product/[barcode].json

Contoh: https://world.openpetfoodfacts.org/api/v0/product/3600550578041.json

  • API V1- Dokumentasi lama tersedia di Wiki (Catatan penting: halaman ini dibagi menjadi subhalaman untuk BACA dan TULIS).
  • API V2 - Komprehensif tetapi tidak sesuai dengan OpenAPI - Berisi kasus penggunaan untuk menampilkan produk, dan memungkinkan pengguna Anda menambahkan data yang diperlukan untuk mendapatkan Nutri-Score, klasifikasi NOVA, zat tambahan, dan Eco-Score.
  • API V2 - dokumentasi OpenAPI baru - Sedang dalam Proses
  • API V3 - dokumentasi OpenAPI baru - Sedang dalam Proses

Anda dipersilahkan untuk menggunakan API untuk kasus - kasus produksi, selama 1 pemanggilan API = 1 pemindaian nyata oleh pengguna. Upaya apapun untuk mengikis basis data menggunakan API akan sangat mungkin diblokir, karena ekspor harian lengkap tersedia di halaman ini.

Juga, mengirim header HTTP dengan pemanggilan API Anda adalah cara yang baik untuk menghubungi kami, jika ada masalah yang muncul.

Jika Anda tidak yakin tentang apapun, atau memiliki pertanyaan (tidak ada pertanyaan konyol), jangan sungkan untuk menanyakannya di Slack :-)

Aplikasi seluler Android dan iPhone (perlu bantuan!)

Kode untuk aplikasi seluler Open Pet Food Facts yang saat ini digunakan tersedia di GitHub (Android). Kami membutuhkan bantuan untuk mengadaptasi kode Flutter yang baru untuk makanan hewan peliharaan. Aplikasi ini memungkinkan pengguna untuk memindai kode batang produk, melihat informasi produk, dan mengambil serta mengirim gambar dan data untuk produk yang hilang.

Aplikasi untuk Android dikembangkan di Kotlin.

Aplikasi klasik untuk iOS tidak pernah diadaptasi atau digunakan untuk makanan hewan peliharaan.

Aplikasi lintas platform baru ini dikembangkan di Flutter/Dart.

Anda dipersilahkan untuk menggunakan kodenya kembali (ini bersifat sumber terbuka) dan membantu kami meningkatkannya, untuk semua orang, di seluruh planet ini.
