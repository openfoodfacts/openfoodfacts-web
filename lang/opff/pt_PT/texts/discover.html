Uma base de dados de produtos para alimentação de animais de estimação

O Open Pet Food Facts é uma base de dados de produtos para animais de estimação com ingredientes, valores nutricionais e todas as informações que conseguimos encontrar nos rótulos dos produtos.

Feito por todos

A Open Pet Food Facts é uma associação sem fins lucrativos composta por voluntários.

Os donos de animais de estimação como você podem adicionar produtos de todo o mundo utilizando a aplicação Open Food Facts para Android ou iPhone ou a sua câmara para digitalizar códigos de barras e enviar fotografias dos produtos e dos seus rótulos.

Descarregue a aplicação para começar a digitalizar ração para animais de estimação.

Para todos

Os dados sobre os alimentos para animais de companhia são de interesse público e devem ser transparentes. A base de dados completa é publicada como dados abertos e pode ser reutilizada por qualquer pessoa e para qualquer utilização. Veja as reutilizações interessantes ou faça suas próprias!

Faça melhores escolhas em relação à alimentação do seu animal de estimação.

Conhecimento é poder. Conhecimento sobre alimentação animal é poder para alimentar melhor os seus animais de estimação!

  • Descodifique rótulos

    Ingredientes, aditivos e valores nutricionais dos alimentos para animais de estimação... O site Open Pet Food Facts ajuda-o a compreender as informações detalhadas nos rótulos dos produtos.

  • Encontre produtos

    Nosso formulário de busca inclui dezenas de critérios para que você possa encontrar produtos alimentares para animais de estimação que correspondam às necessidades de seus animais de estimação.

  • Compare e escolha

    Sabe quais as rações para cães que têm mais proteína? Que alimentos para gatos são isentos de cereais? Pode comparar produtos de diferentes marcas e tomar decisões informadas para a saúde dos seus animais de estimação.

  • Explore e descubra

    Tem curiosidade sobre alimentos para animais de estimação? Explore os produtos por critérios como marcas, categorias, rótulos, espécies (cães, gatos, etc.) e ingredientes. Descubra o que realmente contém a comida do seu animal de estimação.

Alimentos para animais de estimação que promovem a inovação e a ciência.

Os nossos dados podem ser reutilizados para educação, novos produtos e serviços, pesquisas e muito mais!

A base de dados Open Pet Food Facts é um conjunto de dados abertos publicado sob a Open Database License. Qualquer pessoa pode utilizá-lo para qualquer finalidade - comercial ou não comercial - desde que a fonte dos dados seja atribuída e as melhorias na base de dados sejam partilhadas de forma semelhante.

Educação

Aprenda sobre nutrição animal e compreenda o que constitui uma dieta saudável para o seu cão, gato ou outros animais de estimação. Os dados abertos ajudam os donos de animais de estimação a tomar decisões informadas.

Produtos e serviços

Os dados do Open Pet Food Facts podem ser reutilizados tanto em projetos não comerciais como comerciais, desde que as melhorias efetuadas na base de dados sejam também partilhadas.

Ciência

Os dados do Open Pet Food Facts podem ser utilizados por veterinários, nutricionistas de animais de estimação e investigadores que estudam a nutrição e a saúde animal. Ajude a contribuir para uma melhor ciência na alimentação animal!
