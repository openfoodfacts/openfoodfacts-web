Open Pet Food Facts

Descobrir

A Open Pet Food Facts é uma base de dados sobre alimentos para animais de estimação criada por todos e para todos. Pode utilizá-lo para fazer melhores escolhas para o seu animal de estimação e, como se trata de dados abertos, qualquer pessoa pode reutilizá-los para qualquer finalidade.

Contribuir

- O Open Pet Food Facts é um projeto não lucrativo desenvolvido por milhares de voluntários em todo o mundo. Pode começar a contribuir adicionando um produto alimentar para animais de estimação com a nossa aplicação para iPhone ou Android, e temos muitos projetos interessantes para os quais pode contribuir de várias maneiras diferentes.
