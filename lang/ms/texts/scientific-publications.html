Penerbitan Saintifik

Fakta Makanan Terbuka dalam penyelidikan: penerbitan, petikan dan sumber untuk penyelidik

Sejak tahun 2012, Open Food Facts telah menjadi sumber yang berharga bagi para penyelidik di seluruh dunia yang menggunakan data makanan kami untuk kajian mengenai pemakanan, alam sekitar, kesihatan awam dan banyak bidang lain. Pangkalan data terbuka kami telah dipetik dalam ratusan kertas saintifik.

Bidang Penyelidikan Utama

Data kami menyokong pelbagai domain penyelidikan

Pemakanan & Kesihatan Awam

Para penyelidik menggunakan pangkalan data kami untuk mengkaji kualiti pemakanan, komposisi makanan dan intervensi kesihatan awam. Data Fakta Makanan Terbuka telah memainkan peranan penting dalam penyelidikan dan penilaian Nutri-Score.

Kesan Alam Sekitar

Kajian tentang jejak karbon, Eco-Score dan kelestarian alam sekitar produk makanan mendapat manfaat daripada pangkalan data maklumat produk kami yang komprehensif.

Pemprosesan Makanan

Penyelidikan mengenai makanan ultra-proses menggunakan klasifikasi NOVA bergantung pada senarai ramuan terperinci dan maklumat pemprosesan kami.

Sains Komputer & AI

Pembelajaran mesin, pengecaman imej, pemprosesan bahasa semula jadi dan projek sains data memanfaatkan set data dan imej terbuka kami.

Tingkah Laku Pengguna

Kajian yang mengkaji pilihan makanan, kesan pelabelan dan ketelusan dalam industri makanan menggunakan data Fakta Makanan Terbuka.

Dasar Makanan

Pembuat dasar dan penyelidik menganalisis peraturan makanan, keperluan pelabelan dan amalan industri menggunakan pangkalan data telus kami.

Sumber untuk Penyelidik

Cara menggunakan dan memetik Fakta Makanan Terbuka

Mengakses Data

Open Food Facts menyediakan pelbagai cara untuk mengakses data kami:

  • Eksport pangkalan data: Muat turun lambakan pangkalan data lengkap dalam pelbagai format. Lihat halaman Data kami untuk butiran lanjut.
  • API: Buat pertanyaan tentang pangkalan data kami secara pengaturcaraan melalui API komprehensif kami. Dokumentasi API
  • Antara muka carian: Terokai produk terus di laman web kami dengan keupayaan carian dan penapisan lanjutan.

Memetik Fakta Makanan Terbuka

Jika anda menggunakan Fakta Makanan Terbuka dalam penyelidikan anda, sila petiknya dengan sewajarnya. Berikut ialah format petikan yang dicadangkan:

Petikan laman web:

Fakta Makanan Terbuka. (Tahun). Fakta Makanan Terbuka - Dunia. Diperoleh daripada https://world.openfoodfacts.org


Petikan set data:

Fakta Makanan Terbuka. (Tahun). Pangkalan Data Fakta Makanan Terbuka [Set data]. Diperoleh daripada https://world.openfoodfacts.org/data

Pelesenan

Data kami terbuka dan boleh diguna semula secara bebas:

  • Pangkalan data ini tersedia di bawah Lesen Pangkalan Data Terbuka (ODbL)
  • Kandungan pangkalan data berada di bawah Lesen Kandungan Pangkalan Data (DbCL)
  • Imej produk tersedia di bawah Creative Commons Attribution ShareAlike (CC BY-SA 3.0)

See our Terms and conditions of use and re-use for complete details.

Data Quality

Sebagai pangkalan data sumber ramai, kualiti data boleh berbeza-beza. Apabila menggunakan Fakta Makanan Terbuka untuk penyelidikan:

  • Pertimbangkan untuk menapis mengikut skor kelengkapan atau penunjuk kualiti data
  • Sahkan titik data kritikal jika ketepatan adalah penting
  • Berhati-hati dengan bias geografi dan temporal dalam liputan data kami
  • Hubungi kami jika anda memerlukan bantuan dengan penilaian kualiti data untuk penyelidikan anda

Bekerjasama dengan Kami

Kami mengalu-alukan kerjasama dengan pasukan penyelidikan:

  • Keperluan data: Beritahu kami jika anda memerlukan data khusus atau penambahbaikan pada struktur pangkalan data kami
  • Kumpulan imbasan: Atur kempen pengumpulan data yang disasarkan untuk rantau penyelidikan atau kategori produk anda
  • Perkongsian keputusan: Kongsikan penemuan anda dengan komuniti kami dan bantu tingkatkan pengetahuan
  • Pembangunan alat: Bekerjasama dalam menghasilkan alat yang memberi manfaat kepada penyelidik dan pengguna

Hubungi kami di contact@openfoodfacts.org untuk membincangkan peluang kerjasama.
