Η νοημοσύνη πίσω από τις εφαρμογές Open Food Facts και API

Το Open Food Facts συλλέγει δεδομένα χρησιμοποιώντας ένα WRITE API, το επεξεργάζεται και το καθιστά διαθέσιμο χρησιμοποιώντας ένα READ API και μια διαδικτυακή έκδοση. Υπολογισμός των Nutri-Score, ομάδων NOVA, εξαγωγή προσθέτων, ανεξάρτητα από τη γλώσσα, τα συνώνυμα ή τα πιθανά ορθογραφικά λάθη. Αυτό είναι μόνο ένα μικρό κλάσμα των θεμάτων που καλύπτονται από την ομάδα διακομιστών, για να διασφαλιστεί η διαφάνεια των τροφίμων σε παγκόσμια κλίμακα.

Συνεισφέρετε στον Πυρήνα χρησιμοποιώντας Perl

Θέλετε να μάθετε μια νέα γλώσσα; Ο διακομιστής είναι κυρίως γραμμένος σε Perl. Είτε είστε έμπειρος εθισμένος στην Perl είτε μόλις πιάσατε την Perl για εργασίες διαχείρισης συστήματος, υπάρχει ένα γιγάντιο σφάλμα και μέντορες που θα σας καθοδηγήσουν. Και επίσης περισσότεροι από ένας τρόποι για να το κάνετε ;-)

Δημιουργήστε χρήσιμα εργαλεία σε οποιαδήποτε γλώσσα

Το HungerGames είναι το παράδειγμα ενός μικρού παιχνιδιού γραμμένο σε react που έχει θετικό αντίκτυπο στο Open Food Facts. Το LanguageSwitcher, ένα άλλο εργαλείο, είναι γραμμένο σε Python. Δικό σας το εργαλείο, δική σας γλώσσα (παρακαλώ αποφύγετε το Assembler ;-))
