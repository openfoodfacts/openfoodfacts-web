Συνεισφέρετε στο Open Food Facts

Το Open Food Facts υπάρχει χάρη στους πολίτες που δίνουν το χρόνο και το ταλέντο τους σε πολλές διαφορετικές δραστηριότητες. Είμαστε μια μεγάλη διεθνής οικογένεια που δίνει έναν μοναδικό χαρακτήρα σε αυτό το όμορφο έργο, φτιαγμένο για όλους, από όλους! 🌍✨

Για να διευκολυνθεί η συμμετοχή των συντελεστών μας και να επιτραπεί στο έργο να αναπτυχθεί με πιο δυναμικό τρόπο, δημιουργούμε μια Ομάδα Δεξιοτήτων Συντελεστών, καθώς και ομάδες εθελοντών ανά θέμα.

✅ Στόχος είναι να ταιριάξετε τις δεξιότητες και τα ενδιαφέροντά σας με τις ανάγκες του έργου.

Είτε είστε ήδη ένας πολύ εμπλεκόμενος συνεισφέρων ή μόλις ήρθατε και θέλετε να συμμετάσχετε, σας καλούμε να συμπληρώσετε την παρακάτω φόρμα. (Όλα τα προσωπικά δεδομένα που εισάγετε θα διατηρηθούν εμπιστευτικά και θα χρησιμοποιηθούν μόνο από τη μόνιμη ομάδα για να επικοινωνήσει μαζί σας).

Βοηθήστε να μεταφραστεί το Open Food Facts

Ζητείται βοήθεια από ειδικούς

Η ανάπτυξη Open Food Facts απαιτεί επίσης εξειδικευμένη γνώση και εξειδίκευση σε πολλούς διαφορετικούς τομείς:

  • Διαχείριση Έργου

    Έχουμε πάρα πολλές ιδέες και μάλλον έχετε ακόμη περισσότερες, αλλά είναι μια μεγάλη πρόκληση να μπουν προτεραιότητες, να δημιουργηθεί σχέδιο δράσης και να διαχειριστούν προγράμματα όταν ο κάθε συμμετέχων είναι εθελοντής με συνήθως περιορισμένο χρόνο διαθέσιμο.

  • Σχεδίαση

    Χρειαζόμαστε βοήθεια για να δημιουργήσουμε μια καλύτερη εμπειρία χρήστη στον ιστότοπο Open Food Facts και στην εφαρμογή για κινητά, να βελτιώσουμε τον σχεδιασμό τους, να δημιουργήσουμε εντυπωσιακό υλικό παρουσίασης κλπ.

    Γίνετε μέλος στην σχεδιαστική μας προσπάθεια

  • Ανάπτυξη

    Έχουμε πολλή ανάπτυξη να κάνουμε. Στο backend Open Food Facts (Perl και MongoDB), στο API (JSON), στον ιστότοπο (πρότυπο HTML5, JS, Foundation), στην εφαρμογή μας για κινητά Flutter για iOS και Android, αλλά και σε δημιουργία νέων φοβερών επαναχρησιμοποιήσεων κλπ. Έχουμε έργα SDK σε πολλές γλώσσες προγραμματισμού για να διασφαλίσουμε ότι ο καθένας μπορεί να επαναχρησιμοποιήσει και να συνεισφέρει στα Open Food Facts, σε οποιαδήποτε γλώσσα. Έχουμε επίσης μια αυξανόμενη προσπάθεια Τεχνητής Νοημοσύνης για να απλοποιήσουμε τις εργασίες συνεισφοράς. Ο κώδικάς μας είναι στο GitHub. Εξερευνήστε τη λίστα έργων στο GitHub

  • Δημιουργία κοινοτήτων

    Πρέπει να δημιουργήσουμε τοπικές κοινότητες σε όλες τις χώρες και ταυτόχρονα να τις ενώσουμε σε παγκόσμιο επίπεδο.

    Είναι πολύ δύσκολο να δημιουργήσετε μια τοπική κοινότητα σε μια χώρα χωρίς να μένετε εκεί, επομένως η βοήθειά σας για να βρείτε τους πρώτους με πολλά κίνητρα συμμετέχοντες είναι απαραίτητη. Είστε ένας από αυτούς;

  • Επικοινωνία

    Δεν είμαστε σίγουροι τι να γράψουμε εδώ, θα μπορούσατε να βοηθήσετε;

    Πιο σοβαρά, υπάρχουν πολλά ωραία πράγματα που κάνουν οι χρήστες, οι συνεισφέροντες και οι επαναχρησιμοποιητές με τα Open Food Facts, θα ήταν υπέροχο να μάθουν περισσότεροι άνθρωποι γι' αυτό. Ειδικότερα, χρειαζόμαστε βοήθεια για τις δημόσιες σχέσεις και σχέσεις με τα μέσα ενημέρωσης.

  • Ειδικά έργα

    Υπάρχουν πολλές ενδιαφέρουσες και πρωτότυπες εφαρμογές ανοιχτών δεδομένων τροφίμων που θα μπορούσαμε να εργαστούμε με κυβερνητικούς οργανισμούς τροφίμων, παραγωγούς τροφίμων, ερευνητές, πανεπιστήμια, σχολεία, ΜΚΟ κλπ. Ίσως έχετε ήδη ιδέες; Αν ναι, παρακαλούμε να μας βοηθήσετε να τους προωθήσουμε.
