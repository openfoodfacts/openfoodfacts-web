Όροι επαναχρησιμοποίησης

Η βάση δεδομένων Open Food Facts είναι διαθέσιμη με την άδεια Open Database License.
Τα μεμονωμένα περιεχόμενα της βάσης δεδομένων είναι διαθέσιμα με την άδεια Περιεχόμενα βάσης δεδομένων. Οι εικόνες προϊόντων
είναι διαθέσιμες με την άδεια Creative Commons Αναφορά Παρόμοιου Χρήσης. Μπορεί να περιέχουν γραφικά στοιχεία που υπόκεινται σε πνευματικά δικαιώματα ή άλλα δικαιώματα, τα οποία ενδέχεται σε ορισμένες περιπτώσεις να αναπαραχθούν (δικαιώματα προσφοράς ή ορθή χρήση).

Διαβάστε Όρους και προϋποθέσεις χρήσης και επαναχρησιμοποίησης πριν χρησιμοποιήσετε ξανά τα δεδομένα.

Πείτε μας για την επαναχρησιμοποίησή σας

Μας ενδιαφέρει πολύ να μάθουμε για ποιο σκοπό χρησιμοποιούνται τα δεδομένα Open Food Facts. Δεν είναι υποχρεωτικό, αλλά θα το εκτιμούσαμε πολύ αν μας πείτε για τις επαναχρησιμοποιήσεις σας ώστε να τις μοιραστούμε με την κοινότητα Open Food Facts. Μπορείτε επίσης να συμπληρώσετε αυτήν τη φόρμα για να έχετε την ευκαιρία να αναδείξετε την εφαρμογή σας μέσω εμάς.

Εξαγωγές δεδομένων Open Food Facts

Οι καταχωρήσεις βάσεων δεδομένων και οι εξαγωγές δημιουργούνται κάθε νύχτα. Η βάση δεδομένων είναι πολύ μεγάλη, επομένως μπορείτε να βρείτε διαφορετικούς τρόπους χρήσης των δεδομένων, ανάλογα με τον τρόπο εξαγωγής.

Πληροφορίες σχετικά με τα διαφορετικά πεδία για τη καταχώρηση MongoDB και εξαγωγές CSV είναι διαθέσιμες στη διεύθυνση https://world.openfoodfacts.org/data/data-fields.txt

Καταχώρηση MongoDB

Τα δεδομένα για όλα τα προϊόντα είναι διαθέσιμα σε μια καταχώρηση βάσης δεδομένων MongoDB.

Σύνδεσμος
https://static.openfoodfacts.org/data/openfoodfacts-mongodbdump.gz
sha256sum
https://static.openfoodfacts.org/data/gz-sha256sum
md5sum
https://static.openfoodfacts.org/data/gz-md5sum

Delta Export

Παρέχονται ημερήσιες εξαγωγές δέλτα για τις προηγούμενες 14 ημέρες. Η λίστα των διαθέσιμων αρχείων δέλτα βρίσκεται στη διεύθυνση https://static.openfoodfacts.org/data/delta/index.txt.

Κάθε γραμμή αντιπροσωπεύει ένα αρχείο που είναι διαθέσιμο στο https://static.openfoodfacts.org/data/delta/{filename}.

Το όνομα αρχείου περιέχει χρονική σήμανση UNIX της πρώτης και της τελευταίας αλλαγής που περιέχεται στο αρχείο JSON, έτσι ώστε τα αρχεία δέλτα να μπορούν να εισαχθούν (μετά την εξαγωγή) με mongoimport σε αλφαβητική σειρά.

Λάβετε υπόψη ότι λόγω της φύσης του mongoexport, τα αρχεία δέλτα δεν μπορούν να σας ενημερώσουν για τα διαγραμμένα προϊόντα. Για να αφαιρέσετε τα διαγραμμένα προϊόντα από τη βάση δεδομένων σας, θα χρειαστεί να εισαγάγετε την πλήρη καταχώρηση MongoDB.

Εξαγωγή δεδομένων JSONL

Ολόκληρη η βάση δεδομένων είναι επίσης διαθέσιμη σε μορφή JSONL (μερικές φορές ονομάζεται LDJSON ή NDJSON) όπου κάθε γραμμή είναι ένα αντικείμενο JSON. Αντιπροσωπεύει τα ίδια δεδομένα με την εξαγωγή σε MongoDB. Το αρχείο είναι συμπιεσμένο με gzip.

Σύνδεσμος
https://static.openfoodfacts.org/data/openfoodfacts-products.jsonl.gz

Ένας κατάλληλος τρόπος για την εκμετάλλευση της βάσης δεδομένων είναι η χρήση του DuckDB, ενός αναλυτικού εργαλείου που έχει σχεδιαστεί για την επεξεργασία μεγάλου όγκου δεδομένων σε κλάσματα δευτερολέπτων. Μπορείτε να διαβάσετε την ανάρτησή μας στο ιστολόγιο όπου σας καθοδηγούμε στην εξερεύνηση και την επεξεργασία της βάσης δεδομένων Open Food Facts με το DuckDB

Εξαγωγή δεδομένων Parquet στο Hugging Face

Μια απλοποιημένη έκδοση της καταχώρησης JSONL είναι επίσης διαθέσιμη σε μορφή Parquet. Κατά τη μετατροπή, φιλτράραμε στήλες που περιέχουν διπλότυπες πληροφορίες, χρησιμοποιούνται για εσωτερικό εντοπισμό σφαλμάτων ή είναι απλώς περιττές για τους χρήστες.

Η μορφή Parquet έχει αποδειχθεί εύχρηστη:

  • Τα δεδομένα οργανώνονται ανά στήλη και όχι κατά σειρά, γεγονός που εξοικονομεί χώρο αποθήκευσης και επιταχύνει τα ερωτήματα αναλυτικών στοιχείων, δηλαδή μπορείτε να επιλέξετε μόνο τις στήλες που σας ενδιαφέρουν, βελτιστοποιώντας τις επιδόσεις των ερωτημάτων, ακόμη και σε υπολογιστές μεσαίας κατηγορίας.
  • Ιδιαίτερα αποτελεσματική συμπίεση και αποσυμπίεση δεδομένων, καθιστώντας τα κατάλληλα για την αποθήκευση και την κοινή χρήση μεγάλων συνόλων δεδομένων κάθε είδους,
  • Υποστηρίζει σύνθετους τύπους δεδομένων και προηγμένες εμφωλευμένες δομές δεδομένων.

Το σύνολο δεδομένων είναι διαθέσιμο στο Hugging Face, ένα συνεργατικό οικοσύστημα Μηχανικής Μάθησης όπου οι προγραμματιστές και οι ερευνητές μπορούν να μοιράζονται μοντέλα και σύνολα δεδομένων.

Σύνδεσμος
https://huggingface.co/datasets/openfoodfacts/product-database/resolve/main/food.parquet?download=true

Βρείτε περισσότερες πληροφορίες στο Wiki, συμπεριλαμβανομένων οδηγιών για επαναχρησιμοποίηση δεδομένων και παραδειγμάτων ερωτημάτων για να ξεκινήσετε.

Εξαγωγή δεδομένων CSV

Τα δεδομένα για όλα τα προϊόντα ή ορισμένα από τα προϊόντα μπορούν να ληφθούν σε μορφή CSV (αναγνώσιμα από το LibreOffice, το Excel και πολλά άλλα λογισμικά υπολογιστικών φύλλων) μέσω της φόρμας σύνθετης αναζήτησης.

Σύνδεσμοι
https://static.openfoodfacts.org/data/en.openfoodfacts.org.products.csv.gz (συμπιεσμένο CSV σε μορφή GZIP: ~ 0,9 Gb, μη συμπιεσμένο: ~ 9 Gb)

Η κωδικοποίηση του αρχείου είναι Unicode UTF-8. Ο χαρακτήρας που χωρίζει τα πεδία είναι <tab> (tabulation).

Εξαγωγή δεδομένων RDF

Η βάση δεδομένων είναι επίσης διαθέσιμη σε μορφή RDF. Μπορείτε να διαβάσετε την ανακοίνωση στα γαλλικά.

Ζωντανό JSON και XML API

Ένα JSON API είναι επίσης διαθέσιμο για την ανάγνωση των δεδομένων για ένα προϊόν. Αυτό το API χρησιμοποιείται συγκεκριμένα στην εφαρμογή Open Food Facts για κινητά για iPhone και Android.

Βελτιώνουμε το API με την πάροδο του χρόνου, αλλά προσέχουμε να διατηρήσουμε τη προς τα πίσω συμβατότητα. Παρακαλούμε επικοινωνήστε εάν το χρησιμοποιείτε, ώστε να σας ενημερώσουμε για πιθανές αλλαγές.

URL για ανάγνωση δεδομένων για ένα προϊόν: https://world.openfoodfacts.org/api/v2/product/[barcode].json

Παράδειγμα: https://world.openfoodfacts.org/api/v3/product/737628064502.json

Σημειώστε ότι εάν η έξοδος είναι υπερβολικά μεγάλη, μπορείτε να τη φιλτράρετε - όπως περιγράφεται στην τεκμηρίωση του API.

  • API V3 - new OpenAPI documentation - Work in Progress

Ζητείται βοήθεια για το OpenAPI

Ανυπομονούμε να ολοκληρώσουμε την τεκμηρίωση του API, να προσθέσουμε περιπτώσεις χρήσης και να αυτοματοποιήσουμε τη δημιουργία SDK. Εάν είστε γνώστης του OpenAPI, είστε ευπρόσδεκτοι να βοηθήσετε.

Είστε ευπρόσδεκτοι να χρησιμοποιήσετε το API για περιπτώσεις δημόσιας χρήσης, αρκεί 1 κλήση API = 1 πραγματική σάρωση από έναν χρήστη. Οποιαδήποτε προσπάθεια αντιγραφής της βάσης δεδομένων χρησιμοποιώντας το API είναι πολύ πιθανό να αποκλειστεί, καθώς πλήρεις καθημερινές εξαγωγές είναι διαθέσιμες σε αυτήν ακριβώς τη σελίδα.

Επίσης, η αποστολή μιας κεφαλίδας HTTP με την κλήση API σας είναι ένας καλός τρόπος για να έρθουμε σε επαφή, σε περίπτωση που προκύψει οποιοδήποτε πρόβλημα.

Εάν δεν είστε σίγουροι για κάτι ή έχετε οποιαδήποτε ερώτηση (δεν υπάρχουν χαζές ερωτήσεις), μη διστάσετε να τη ρωτήσετε στο κανάλι #api στο Slack :-)

Πειραματικό XML API

Περιτύλιξη και SDK

Πιθανότατα έχουμε ένα περιτυλιχτή για την αγαπημένη σας γλώσσα προγραμματισμού. Εάν ναι, μπορείτε να το χρησιμοποιήσετε και να το βελτιώσετε. Εάν όχι, μπορείτε να βοηθήσετε στη δημιουργία του.

Θα σας επιτρέψουν να καταναλώνετε δεδομένα, καθώς και να επιτρέπουν στους χρήστες σας να συνεισφέρουν νέα δεδομένα

Γενικές αρχές για να κάνετε τους χρήστες σας χαρούμενους

  • Μπορείτε να αναζητήσετε πληροφορίες σχετικά με προϊόντα, συμπεριλαμβανομένων πολλών χρήσιμων υπολογιστικών τιμών.
  • Εάν δεν μπορείτε να λάβετε τις πληροφορίες για ένα συγκεκριμένο προϊόν, μπορείτε να κάνετε τον χρήστη σας να στείλει φωτογραφίες και δεδομένα, τα οποία στη συνέχεια θα υποβληθούν σε επεξεργασία από το Open Food Facts AI και τους συνεργάτες για να λάβετε το υπολογισμένο αποτέλεσμα που θέλετε να τους εμφανίσετε.
  • Μπορείτε επίσης να εφαρμόσετε την πλήρη ροή ώστε να πάρουν αμέσως το αποτέλεσμα με μερική προσπάθεια από την πλευρά τους.

Εάν οι χρήστες σας δεν αναμένουν άμεσα αποτέλεσμα (π.χ. εφαρμογές για απόθεμα)

  • Υποβάλετε φωτογραφίες (μπροστά/διατροφή/συστατικά): το πιο ανώδυνο πράγμα για τους χρήστες σας
  • Το Open Food Facts AI - το Robotoff - θα δημιουργήσει κάποια δεδομένα από τις φωτογραφίες
  • Με την πάροδο του χρόνου, άλλες εφαρμογές και η κοινότητα Open Food Facts θα καλύψουν τα κενά δεδομένων

Εάν οι χρήστες σας αναμένουν ένα αποτέλεσμα αμέσως (π.χ. εφαρμογές Διατροφής)

  • Υποβάλετε στοιχεία διατροφής + κατηγορία > λάβετε Nutri-Score
  • Υποβάλετε συστατικά > λάβετε την ομάδα NOVA (σχετικά με την υπερεπεξεργασία τροφίμων), πρόσθετα, αλλεργιογόνα, κανονικοποιημένα συστατικά, vegan, χορτοφαγικά…
  • Υποβάλετε κατηγορία + ετικέτες > λάβετε σύντομα το Eco-Score (σχετικό με τις περιβαλλοντικές επιπτώσεις)

Εφαρμογή για κινητά Android και iPhone

Ο κώδικας για την εφαρμογή Open Food Facts είναι διαθέσιμος στο GitHub (Flutter, Kotlin Android και Swift iOS).

Προσπαθούμε να μετατρέψουμε τις καταργημένες βάσεις κώδικα Kotlin και Swift σε Kotlin και Swift SDKs, κάθε βοήθεια ευπρόσδεκτη :-) Η εφαρμογή επιτρέπει στους χρήστες να σαρώνουν τον γραμμωτό κώδικα των προϊόντων, να προβάλλουν τις πληροφορίες του προϊόντος και να τραβούν και να υποβάλλουν φωτογραφίες και δεδομένα για προϊόντα που λείπουν.

Η εφαρμογή επιτρέπει στους χρήστες να σαρώνουν τον γραμμωτό κώδικα των προϊόντων, να προβάλλουν τις πληροφορίες του προϊόντος και να λαμβάνουν και να υποβάλλουν φωτογραφίες και δεδομένα για προϊόντα που λείπουν.

Μπορείτε να χρησιμοποιήσετε ξανά τον κώδικα (είναι ανοιχτού κώδικα) και να μας βοηθήσετε να τον βελτιώσουμε, για όλους, σε όλο τον πλανήτη.
