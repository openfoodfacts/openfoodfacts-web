Una base de datos de productos alimenticios

Open Food Facts es una base de datos de productos alimenticios con ingredientes, alérgenos, datos nutricionales y toda la información que podemos encontrar en las etiquetas de los productos.

Hecho por todos

Open Food Facts es una asociación de voluntarios sin ánimo de lucro.

Más de 100.000 colaboradores como tú han añadido 4 000 000+ productos de 150 países usando nuestra aplicación para Android o iPhone o su cámara para escanear códigos de barras y cargar imágenes de productos y sus etiquetas. Obtén la aplicación para comenzar a escanear y contribuir

Para todos

Los datos sobre los alimentos son de interés público y deben estar abiertos. Toda la base de datos se publica como datos abiertos y puede ser reutilizada por cualquiera y para cualquier uso. ¡Echa un vistazo a las reutilizaciones geniales o crea la tuya propia!

  • Descodificar etiquetas

    Aditivos alimentarios: números E, alérgenos, códigos de trazabilidad... Open Food Facts le ayuda a comprender la letra pequeña de las etiquetas de los productos.

    También evaluamos la calidad nutricional de cada producto.

  • Encontrar productos

    Nuestro formulario de búsqueda incluye docenas de criterios para que pueda encontrar productos que coincidan con sus criterios.

  • Comparar y cambiar

    ¿Sabes qué cereales de desayuno tienen menos grasa y azúcar? Con solo 3 clics, podrás crear un gráfico interactivo para mostrarlo. De esta manera usted puede cambiar a cereales as saludables, y comparando productos de marcas diferentes, anima a los productores a mejorarlos. Alimentos para el pensamiento, la innovación y la ciencia

¡Nuestros datos se pueden reutilizar para el ámbito educativo, la elaboración de nuevos productos y servicios, la investigación y más!

La base de datos de Open Food Facts es información abierta publicada bajo la Open Database License. Cualquier persona lo puede usar para cualquier propósito -comercial o no comercial- si se atribuyen la fuente de los datos y las mejoras a la base de datos son compartidas de forma similar.

Ciencia

Los datos de Open Food Facts son utilizados por científicos en diferentes campos (nutrición, biología, datos abiertos vinculados, ...). ¡Y no necesitas un doctorado para formar parte de la ciencia ciudadana!

Grados de nutrición

El Programa Nacional de Nutrición y Salud de Francia utiliza Open Food Facts para validar la fórmula de su puntuación de calidad nutricional y grados de nutrición.
