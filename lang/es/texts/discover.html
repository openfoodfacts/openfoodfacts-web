Una base de datos de productos cosméticos

Open Beauty Facts es una base de datos de productos cosméticos con ingredientes, alérgenos, información y todos los fragmentos de información que podemos encontrar en las etiquetas de los productos.

Hecho por todos

Open Beauty Facts es una asociación de voluntarios sin fines de lucro.

Más de 1800 colaboradores como usted han agregado 13 000+ productos de 134 países utilizando nuestra aplicación Android, iPhone o Windows Phone o su cámara para escanear códigos de barras y cargar imágenes de productos y sus etiquetas.

Para todos

Los datos sobre cosméticos son de interés público y deben ser abiertos. Echa un vistazo a las geniales reutilizaciones, ¡o haz la tuya! La base de datos completa se publica como datos abiertos y puede ser reutilizada por cualquiera y para cualquier uso.

  • Encontrar productos

    Nuestro formulario de búsqueda incluye docenas de criterios para que pueda encontrar productos que coincidan con sus criterios.

Cosméticos, para la innovación y la ciencia

¡Nuestros datos se pueden reutilizar para el ámbito educativo, la elaboración de nuevos productos y servicios, la investigación y más!

La base de datos Open Beauty Facts es una base de datos abiertos publicados bajo la licencia Open Database. Cualquiera puede usarlo para cualquier propósito, comercial o no comercial, siempre que se atribuya la fuente de los datos y que las mejoras a la base de datos se compartan de manera similar.

Ciencia

Los datos de Open Beauty Facts son utilizados por científicos en diferentes campos (química, biología, datos enlazados y abiertos...). ¡Y no necesitas un doctorado para formar parte de la ciencia ciudadana!

Grados de nutrición

El Programa Nacional de Nutrición y Salud de Francia utiliza Open Food Facts para validar la fórmula de su puntuación de calidad nutricional y grados en nutrición.
