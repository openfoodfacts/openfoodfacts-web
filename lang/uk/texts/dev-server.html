Інтеллект, який допомагає Open Food Facts та API

Open Foods Facts збирає інформацію за допомогою WRITE API, обробляє її та надає доступ використовуючи READ API та веб-версію. Підрахунок Nutri-Score, групи NOVA, вилучення добавок, незалежно від їх мови, синонімів або потенційних орфографічних помилок. Це лише маленька частина тем, які розглядає наша команда серверів, щоб забезпечити прозорість системи харчування на планетарому рівні.

Зробіть свій внесок у ядро за допомогою Perl

Хочеш вивчити нову мову? Сервер в основному закодований у Perl. Якщо ви досвічений користувач Perl, або вперше користуєтесь Perl для виконання завдань адміністратора, є баг вашого розміру та наставники, які допоможуть вам. А також більш, ніж один спосіб виправити помилку ;-)

Створіть корисні інструменти на будь-якій мові

HungerGames - це приклад маленької гри, закодованої у react, що має позитивний вплив на OpenFoodFacts. LanguageSwitcher, інший інструмент, закодований у Piton. Тобто ваш інструмент, ваша мова (будь ласка уникайте Assembler ;-)
